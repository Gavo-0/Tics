Curso de Bootstrap - Clase .table-dark

Aplicación de la clase .table-dark en Bootstrap 5

Al aplicar la clase table-dark junto con table a una tabla HTML, Bootstrap aplica un tema oscuro a toda la tabla. Esto significa que los colores de fondo, bordes y texto se adaptan para ofrecer una apariencia visual más sobria y contrastante, ideal para interfaces modernas o de baja luminosidad.

¿Qué cambios visuales observamos en el navegador?

  • Fondo oscuro para toda la tabla, tanto en el encabezado (<thead>) como en el cuerpo (<tbody>).
  • Texto en color claro (generalmente blanco o gris claro) para asegurar buen contraste.
  • Bordes sutiles en tonos grises oscuros o intermedios, que dividen filas y columnas.
  • Tipografía clara y limpia, manteniendo el estilo de Bootstrap pero adaptada al modo oscuro.

Este resultado es especialmente útil si el diseño del sitio utiliza una paleta oscura o si se busca reducir la fatiga visual en interfaces nocturnas o de administración. 

<table class="table table-dark">
  ...
</table>

Ventajas del uso de .table-dark

  • Accesibilidad visual: Aumenta el contraste para usuarios en entornos con poca luz.
  • Estética moderna: Ideal para dashboards, paneles de control y temas oscuros.
  • Consistencia: Se adapta a otros componentes oscuros de Bootstrap como navbar-dark o bg-dark.

Combinación con otras clases

La clase .table-dark se puede combinar con otras clases para enriquecer la tabla. Algunas combinaciones comunes son:

  1. table-striped: Fondo rayado alterno, aún en tema oscuro, usando grises claros en filas alternas.
    <table class="table table-dark table-striped">...</table>
  2. table-hover: Resalta la fila bajo el cursor, adaptado a tonos claros en modo oscuro.
    <table class="table table-dark table-hover">...</table>
  3. table-bordered: Bordes definidos entre celdas, útil para separar mejor los datos.
    <table class="table table-dark table-bordered">...</table>
  4. table-sm: Menos padding en celdas para mostrar más información en menos espacio.
    <table class="table table-dark table-sm">...</table>

Ejemplo combinado

Puedes combinar todas estas clases para crear una tabla oscura, rayada, con efecto hover, bordes y diseño compacto: 

<table class="table table-dark table-striped table-hover table-bordered table-sm">
  ...
</table>

Uso con tablas responsivas

Si tu tabla tiene muchas columnas, puedes combinarla con el contenedor .table-responsive para deslizarla horizontalmente en pantallas pequeñas: 

<div class="table-responsive">
  <table class="table table-dark table-striped">
    ...
  </table>
</div>

En pantallas pequeñas, este contenedor agrega una barra de desplazamiento horizontal sin romper el diseño.

Conclusión

La clase table-dark de Bootstrap 5 es una herramienta poderosa para dar un estilo profesional y oscuro a las tablas HTML. Combinada con otras clases utilitarias como table-striped, table-hover, table-bordered o table-sm, permite crear tablas atractivas, funcionales y perfectamente adaptadas a diferentes necesidades de diseño y accesibilidad.
