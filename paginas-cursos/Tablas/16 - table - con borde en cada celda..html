Curso de Bootstrap - table-bordered y table-dark

Tablas con bordes completos en Bootstrap 5

Por defecto, una tabla en Bootstrap 5 solo muestra líneas separadoras entre las filas. Si queremos que todas las celdas tengan bordes, simplemente agregamos la clase table-bordered: 

<table class="table table-bordered">
  <thead>
    <tr>
      <th>Titulo 1</th>
      <th>Titulo 2</th>
      <th>Titulo 3</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>Elemento 1,1</td><td>Elemento 1,2</td><td>Elemento 1,3</td></tr>
    <tr><td>Elemento 2,1</td><td>Elemento 2,2</td><td>Elemento 2,3</td></tr>
    <tr><td>Elemento 3,1</td><td>Elemento 3,2</td><td>Elemento 3,3</td></tr>
    <tr><td>Elemento 4,1</td><td>Elemento 4,2</td><td>Elemento 4,3</td></tr>
    <tr><td>Elemento 5,1</td><td>Elemento 5,2</td><td>Elemento 5,3</td></tr>
  </tbody>
</table>

Así veremos en pantalla una tabla en la que todas las celdas aparecen recuadradas:

Titulo 1 Titulo 2 Titulo 3
Elemento 1,1Elemento 1,2Elemento 1,3
Elemento 2,1Elemento 2,2Elemento 2,3
Elemento 3,1Elemento 3,2Elemento 3,3
Elemento 4,1Elemento 4,2Elemento 4,3
Elemento 5,1Elemento 5,2Elemento 5,3

Combinando con table-dark

Si además agregamos la clase table-dark, la tabla tendrá bordes y tema oscuro al mismo tiempo: 

<table class="table table-bordered table-dark">
  ...
</table>
Titulo 1 Titulo 2 Titulo 3
Elemento 1,1Elemento 1,2Elemento 1,3
Elemento 2,1Elemento 2,2Elemento 2,3
Elemento 3,1Elemento 3,2Elemento 3,3
Elemento 4,1Elemento 4,2Elemento 4,3
Elemento 5,1Elemento 5,2Elemento 5,3

De esta forma, la tabla se adapta tanto a diseños claros como oscuros y las celdas quedan perfectamente recuadradas.
