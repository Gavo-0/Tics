Curso de Bootstrap - table-sm

Tablas condensadas con table-sm en Bootstrap 5

Si tenemos que mostrar grandes tablas con datos, puede ser necesario condensar la información para mostrar más filas y columnas en pantalla. Esto se logra agregando la clase table-sm a la tabla, lo que reduce el padding y hace la tabla más compacta.

Ejemplo comparativo

Tabla normal
Titulo 1 Titulo 2 Titulo 3
Elemento 1,1Elemento 1,2Elemento 1,3
Elemento 2,1Elemento 2,2Elemento 2,3
Elemento 3,1Elemento 3,2Elemento 3,3
Elemento 4,1Elemento 4,2Elemento 4,3
Elemento 5,1Elemento 5,2Elemento 5,3
Tabla condensada (table-sm)
Titulo 1 Titulo 2 Titulo 3
Elemento 1,1Elemento 1,2Elemento 1,3
Elemento 2,1Elemento 2,2Elemento 2,3
Elemento 3,1Elemento 3,2Elemento 3,3
Elemento 4,1Elemento 4,2Elemento 4,3
Elemento 5,1Elemento 5,2Elemento 5,3

Código de ejemplo

<table class="table">
  ...
</table>

<table class="table table-sm">
  ...
</table>

Puedes ver claramente la diferencia entre una tabla por defecto y una condensada: la tabla con table-sm ocupa menos espacio vertical, permitiendo mostrar más información en pantalla, ideal para reportes o sistemas administrativos.
