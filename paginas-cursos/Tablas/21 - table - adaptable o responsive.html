Tablas responsivas con table-responsive en Bootstrap 5

Cuando tenemos tablas con muchas columnas y dependiendo del tamaño del dispositivo, puede ser necesario mostrar una barra de desplazamiento horizontal para ver todos los datos de la tabla sin romper el diseño.

Bootstrap 5 resuelve esto mediante las siguientes clases, que deben aplicarse a un <div> contenedor:

  • table-responsive
  • table-responsive-sm
  • table-responsive-md
  • table-responsive-lg
  • table-responsive-xl
  • table-responsive-xxl

Ejemplo básico usando table-responsive: 

<div class="table-responsive">
  <table class="table">
    <thead>
      <tr>
        <th>Titulo 1</th>
        <th>Titulo 2</th>
        <th>Titulo 3</th>
        <th>Titulo 4</th>
        <th>Titulo 5</th>
        <th>Titulo 6</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Elemento 1,1</td>
        <td>Elemento 1,2</td>
        <td>Elemento 1,3</td>
        <td>Elemento 1,4</td>
        <td>Elemento 1,5</td>
        <td>Elemento 1,6</td>
      </tr>
      ...
    </tbody>
  </table>
</div>

Resultado visual:

Titulo 1 Titulo 2 Titulo 3 Titulo 4 Titulo 5 Titulo 6
Elemento 1,1 Elemento 1,2 Elemento 1,3 Elemento 1,4 Elemento 1,5 Elemento 1,6
Elemento 2,1 Elemento 2,2 Elemento 2,3 Elemento 2,4 Elemento 2,5 Elemento 2,6
Elemento 3,1 Elemento 3,2 Elemento 3,3 Elemento 3,4 Elemento 3,5 Elemento 3,6

Si reduces el ancho de la ventana o ves esta página desde un móvil, verás que la tabla agrega automáticamente una barra de desplazamiento horizontal, permitiendo visualizar todas las columnas sin romper el diseño.

Versiones para distintos tamaños de pantalla

Puedes usar variantes como table-responsive-sm, table-responsive-md, etc., para que la tabla solo sea deslizable en dispositivos específicos (por ejemplo, table-responsive-sm solo aplica la barra en pantallas menores a 576px).
