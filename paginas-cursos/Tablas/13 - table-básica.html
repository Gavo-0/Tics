Curso de Bootstrap - Tablas en HTML

Uso de la etiqueta <table> en HTML con Bootstrap 5

La etiqueta <table> de HTML se utiliza para mostrar información estructurada en filas y columnas, es decir, en forma tabular. Cuando se incorpora el framework Bootstrap 5, se habilita una serie de clases que permiten estilizar, organizar y mejorar la presentación de las tablas de forma responsiva, sin necesidad de escribir mucho CSS personalizado.

Estructura básica de una tabla

Una tabla HTML suele estar compuesta por los siguientes elementos:

  • <table>: Elemento raíz de la tabla.
  • <thead>: Sección del encabezado de la tabla.
  • <tbody>: Contenedor del cuerpo de la tabla.
  • <tr>: Fila de la tabla (table row).
  • <th>: Celda de encabezado (table header).
  • <td>: Celda de datos (table data).

Con Bootstrap, basta agregar la clase .table a la etiqueta <table> para aplicar estilos básicos que incluyen separación entre filas, líneas divisorias sutiles, espaciado adecuado y tipografía mejorada. 

<table class="table">
  <thead>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
      <th>Ciudad</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>Ana</td><td>22</td><td>Madrid</td></tr>
    <tr><td>Luis</td><td>24</td><td>Barcelona</td></tr>
    <tr><td>Eva</td><td>21</td><td>Valencia</td></tr>
  </tbody>
</table>

Clases adicionales de Bootstrap 5 para tablas

Además de .table, Bootstrap 5 ofrece una variedad de clases que se pueden aplicar para mejorar o cambiar el comportamiento de la tabla:

  1. .table-striped: Agrega un color de fondo alterno a las filas del cuerpo de la tabla para mejorar la legibilidad. 
    <table class="table table-striped">...</table>
  2. .table-bordered: Agrega bordes a todas las celdas de la tabla. 
    <table class="table table-bordered">...</table>
  3. .table-hover: Aplica un efecto de resaltado cuando el usuario pasa el cursor sobre las filas. 
    <table class="table table-hover">...</table>
  4. .table-sm: Reduce el espaciado entre celdas para obtener una versión más compacta. 
    <table class="table table-sm">...</table>
  5. .table-dark y .table-light: Permiten aplicar temas oscuros o claros a la tabla. 
    <table class="table table-dark">...</table>
<table class="table table-light">...</table>
  6. Combinación de clases: Es posible combinar múltiples clases para lograr una apariencia más completa: 
    <table class="table table-bordered table-striped table-hover table-sm">...</table>

Tablas responsivas

Para que una tabla sea responsiva (que se pueda deslizar horizontalmente en pantallas pequeñas), se puede encerrar dentro de un contenedor con la clase .table-responsive. Esto es útil especialmente cuando se tienen muchas columnas. 

<div class="table-responsive">
  <table class="table">
    ...
  </table>
</div>

En pantallas pequeñas, este contenedor agregará una barra de desplazamiento horizontal sin romper el diseño general del sitio.

Accesibilidad y buenas prácticas

  • Usa <th> para los encabezados de columnas. Si las filas también tienen encabezado, puedes usar <th scope="row">.
  • Asegúrate de que el contenido sea legible en todos los tamaños de pantalla.
  • Añadir títulos descriptivos en las tablas y usar etiquetas semánticas mejora la accesibilidad.
