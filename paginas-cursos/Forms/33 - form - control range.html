Curso de Bootstrap - Control Range

Control de rango en formularios (form-range)

El control range permite seleccionar un valor numérico dentro de un rango definido, mediante una barra deslizante. Bootstrap 5 aporta la clase form-range para asegurar un diseño uniforme, moderno y responsive en todos los navegadores.
Ventajas:
  • Interfaz visual y fácil de usar para valores numéricos.
  • Permite definir valores mínimo, máximo y el incremento (step).
  • El estilo form-range asegura compatibilidad visual entre navegadores y dispositivos.
  • Puede combinarse con etiquetas o mostrar el valor en tiempo real.
  • Ideal para seleccionar cantidades, ajustar volúmenes, brillo, porcentajes, etc.

Ejemplo básico de control range

<form>
  <label for="rango1" class="form-label">Ejemplo de control rango</label>
  <input type="range" class="form-range" id="rango1">
</form>

Configuración de valores (min, max, step)

Puedes personalizar el rango y precisión del control usando min, max y step. El valor inicial se establece con value. 
<input type="range" class="form-range" id="rango2" min="0" max="5" step="0.5" value="2.5">

Mostrar el valor seleccionado en tiempo real

Es común mostrar el valor actual del range junto a la barra usando JavaScript. Así, el usuario sabe exactamente qué valor está seleccionando. 
<label for="rango3" class="form-label">Volumen: 
  <span id="valorRange3">50</span>%
</label>
<input type="range" class="form-range" id="rango3" min="0" max="100" value="50"
  oninput="document.getElementById('valorRange3').textContent = this.value">

Deshabilitar el control range

Puedes desactivar el control range agregando el atributo disabled. El usuario no podrá modificar el valor y el control aparece atenuado. 
<input type="range" class="form-range" id="rango4" min="1" max="10" value="6" disabled>

Combinar range con texto de ayuda y brillo de toda la pantalla

Puedes utilizar form-text para mostrar ayuda o instrucciones debajo del control.
Ahora este control de brillo ajusta el brillo de toda la pantalla usando filter: brightness con una capa overlay para un efecto uniforme. 
<label for="rango5" class="form-label">
  <i class="bi bi-brightness-high icon-brillo"></i>Brillo: 
  <span id="brillo-valor" class="range-value">80</span>%
</label>
<input type="range" class="form-range mb-1" id="rango5" min="10" max="100" value="80"
  oninput="setBrilloPantalla(this.value)">
<div class="form-text">Elige un valor entre 10 y 100. Cambia el brillo de toda la pantalla.</div>
Elige un valor entre 0 y 100. Cambia el brillo de toda la pantalla.
Tips adicionales:
  • Puedes usar form-range en formularios horizontales o en sistemas de grillas.
  • Utiliza min, max y step según la lógica de tu aplicación.
  • Para accesibilidad, acompaña el range de una etiqueta <label> clara y precisa.
  • Combina form-range con otros controles para interfaces más ricas (por ejemplo: valores de volumen, brillo, porcentajes, etc).
Nota: El brillo se aplica a <html> y una capa overlay para que toda la ventana responda y el oscurecimiento sea absolutamente uniforme.
