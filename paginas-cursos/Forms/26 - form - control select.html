Curso de Bootstrap - Control Select

Formularios: Control select

Para aplicar Bootstrap 5 a un control select solo debes agregar la clase form-select. Puedes usarla tanto para selección única como múltiple. También existen variantes para modificar el tamaño del control.

Ejemplo de Select simple y múltiple

<form>
  <label for="control1">Seleccione un elemento</label> 
  <select class="form-select mb-3" id="control1" name="control1"> 
    <option>1</option> 
    <option>2</option>
    <option>3</option> 
    <option>4</option> 
    <option>5</option> 
  </select>

  <label for="control2">Seleccione varios elementos</label> 
  <select multiple class="form-select mb-3" id="control2" name="control2"> 
    <option>1</option> 
    <option>2</option>
    <option>3</option> 
    <option>4</option> 
    <option>5</option> 
  </select>

  <button type="submit" class="btn btn-primary">Enviar</button>
</form>

Selects grandes y pequeños

Puedes modificar la dimensión del control select agregando las clases form-select-lg (grande) o form-select-sm (pequeño). 
<select class="form-select form-select-sm mb-3" id="smallSelect" name="smallSelect">
  <option>1</option>
  <option>2</option>
  <option>3</option>
  <option>4</option>
  <option>5</option>
</select>
