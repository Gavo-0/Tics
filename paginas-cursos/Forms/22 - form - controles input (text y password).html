Curso de Bootstrap - Formularios y Márgenes

Formularios en Bootstrap 5

Bootstrap 5 facilita la creación de formularios responsivos y modernos utilizando clases utilitarias. Los elementos input y label se agrupan en un <div> para un mejor control visual y de espaciado, aprovechando clases como form-control, form-label y mb-3 para el diseño y la separación.

Ejemplo: Formulario de Login

<form>
  <div class="mb-3">
    <label for="nombre" class="form-label">Ingrese su nombre:</label>
    <input type="text" class="form-control" id="nombre" name="nombre">
  </div>
  <div class="mb-3">
    <label for="clave" class="form-label">Ingrese su clave:</label>
    <input type="password" class="form-control" id="clave" name="clave">
  </div>
  <button type="submit" class="btn btn-primary">Confirmar</button>
</form>
Al aplicar form-label a label e form-control a input, obtienes un diseño claro, con inputs que ocupan el 100% del ancho. La clase mb-3 agrega margen inferior para separar los controles.

Clases de margen en Bootstrap 5

  • mb-* — margin bottom: margen inferior (mb-1, mb-2, ..., mb-5, mb-auto).
  • mt-* — margin top: margen superior (mt-1, ..., mt-5, mt-auto).
  • ms-* — margin start: margen izquierdo (LTR) (ms-1, ..., ms-5, ms-auto).
  • me-* — margin end: margen derecho (LTR) (me-1, ..., me-5, me-auto).
Resumen:
m = margin, top, bottom, start (izquierda), end (derecha). El número indica la cantidad de margen, siendo 1 el menor y 5 el mayor. auto ajusta automáticamente el margen.

Consejos prácticos

  • ✔️ Usa mb-3 en cada campo para separar visualmente los controles.
  • ✔️ Aprovecha form-label y form-control en todos los formularios.
  • ✔️ Combina las clases de margen según el diseño que desees.
  • ❌ No añadas márgenes manuales con estilos en línea, usa las utilidades de Bootstrap.
