Curso de Bootstrap - Control Range

Ejemplos con input-group en Bootstrap 5

La clase input-group permite asociar a un control de formulario un texto o incluso otro control (como un botón) a un lado o a ambos lados del mismo. Esto es útil para mostrar unidades, prefijos, sufijos o acciones asociadas al campo de entrada.
Prefijo de texto (icono @ a la izquierda)
@ 
<div class="input-group mb-3">
  <span class="input-group-text">@</span>
  <input type="text" class="form-control" placeholder="nombre de usuario en el sitio">
</div>
Sufijo de texto (dominio a la derecha)
@gmail.com 
<div class="input-group mb-3">
  <input type="text" class="form-control" placeholder="usuario">
  <span class="input-group-text" id="texto2">@gmail.com</span>
</div>
Botón junto al input
<div class="input-group">
  <button class="btn btn-primary" type="button" id="boton1">Verificar</button>
  <input type="text" class="form-control" placeholder="Nombre de usuario nuevo">
</div>
Tip: Puedes combinar varios elementos en un input-group, como iconos, selectores, botones y más, para crear entradas ricas y contextualizadas.
