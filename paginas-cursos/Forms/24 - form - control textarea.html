Curso de Bootstrap - Textarea

Formularios: Control textarea

El control textarea se utiliza cuando necesitamos ingresar varias líneas de texto. En Bootstrap 5, agrega la clase form-control para obtener el diseño adecuado y elige la cantidad de filas visibles con el atributo rows.

Ejemplo de formulario con textarea

<form>
  <div class="mb-3"> 
    <label for="comentarios" class="form-label">Deje aquí sus comentarios</label>
    <textarea class="form-control" rows="5" id="comentarios" name="comentarios"></textarea>
  </div> 
  <button type="submit" class="btn btn-primary">Enviar</button>      
</form>
Nota: El atributo cols no afecta el ancho del textarea cuando se utiliza form-control en Bootstrap 5. Si necesitas controlar el ancho, utiliza utilidades de grid (.row, .col) o utilidades de tamaño.
