Formularios: Etiquetas flotantes .form-floating

Bootstrap 5 permite mostrar etiquetas que "flotan" sobre los controles del formulario, animándose cuando el usuario hace foco o escribe. Solo necesitas la clase form-floating y definir placeholder en el input o textarea.

Ejemplo de login con etiquetas flotantes

<form>
  <div class="mb-3 form-floating">        
    <input type="text" class="form-control" id="nombre" name="nombre" placeholder="usuario">        
    <label for="nombre">Usuario:</label>
  </div>
  <div class="mb-3 form-floating">
    <input type="password" class="form-control" id="clave" name="clave" placeholder="clave">
    <label for="clave">Clave:</label>
  </div>
  <button type="submit" class="btn btn-primary">Confirmar</button>
</form>
Nota: Para que .form-floating funcione correctamente, el input debe tener un placeholder (aunque su valor puede estar vacío, pero debe existir).
También puedes usarlo en textarea: 
<div class="form-floating mb-3">
  <textarea class="form-control" placeholder="Escribe tu mensaje aquí" id="mensaje" style="height: 100px"></textarea>
  <label for="mensaje">Mensaje</label>
</div>
