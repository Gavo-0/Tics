Formularios: Control de archivos (input file)

Bootstrap 5 permite mejorar los controles de tipo file usando la clase form-control. Puedes crear campos para selección de archivos, habilitar selección múltiple o deshabilitar el control fácilmente.

Ejemplo de formulario con input file

<form>
  <div class="mb-3">
    <label for="archivo1" class="form-label">Control File por defecto</label>
    <input class="form-control" type="file" id="archivo1">
  </div>
  <div class="mb-3">
    <label for="archivo2" class="form-label">Selección de múltiples archivos</label>
    <input class="form-control" type="file" id="archivo2" multiple>
  </div>
  <div class="mb-3">
    <label for="archivo3" class="form-label">Control File deshabilitado</label>
    <input class="form-control" type="file" id="archivo3" disabled>
  </div>
</form>
Notas importantes:
  • Para seleccionar múltiples archivos, usa el atributo multiple:
  • <input class="form-control" type="file" id="archivo2" multiple>
  • Para deshabilitar el campo, usa disabled:
  • <input class="form-control" type="file" id="archivo3" disabled>
