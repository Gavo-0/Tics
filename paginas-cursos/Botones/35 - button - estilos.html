Botones en Bootstrap 5: Clases, colores y significados

Bootstrap 5 propone utilizar las etiquetas <button>, <a> o <input> y define una serie de clases para crear botones de diferentes colores y significados.
Siempre se debe asignar la clase btn y alguna de las clases de color para definir el tipo de botón.

  • btn-primary: Es un botón que se destaca entre un conjunto de botones.
  • btn-secondary: Es un botón no tan importante como btn-primary.
  • btn-success: Se usa para indicar una acción exitosa.
  • btn-info: Botón para información.
  • btn-warning: Advierte que debemos tener cuidado con la acción.
  • btn-danger: Indica una acción peligrosa (por ejemplo, borrar una cuenta).
  • btn-dark: Botón oscuro.
  • btn-light: Botón claro.
  • btn-link: Convierte el botón en un enlace, de menor importancia visual.
