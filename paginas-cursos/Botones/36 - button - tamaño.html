Tamaños de botones en Bootstrap 5

Además de los estilos de color, Bootstrap 5 nos permite definir tres tamaños de botones con clases adicionales:
btn-sm para botones pequeños, btn-lg para botones grandes y el tamaño estándar (sin clase extra).

Clase Descripción
btn-sm Botón pequeño
(sin clase) Botón tamaño normal
btn-lg Botón grande

Ejemplo visual

<button type="button" class="btn btn-primary btn-sm">btn-sm : tamaño pequeño</button>
<button type="button" class="btn btn-primary btn-lg">btn-lg : tamaño grande</button>
<button type="button" class="btn btn-primary">tamaño normal</button>

Recuerda:
- Debemos asignar siempre la clase btn y una clase de color (btn-primary, btn-success, etc).
- Para cambiar el tamaño, solo agrega btn-sm o btn-lg según lo necesites.
