Curso de Bootstrap - Grid y Flex para botones
Ubicación precisa de botones con Grid y Flex en Bootstrap 5
Bootstrap 5 aprovecha utilidades CSS como d-grid y d-flex para distribuir botones según lo que necesitemos:
  • d-grid: Botones apilados verticalmente, ocupando todo el ancho.
  • d-grid d-md-block: Botones apilados en móvil y en fila a partir de tamaño md.
  • d-md-flex justify-content-md-center: Botones centrados desde tamaño md.
  • d-md-flex justify-content-md-end: Botones alineados a la derecha desde tamaño md.
La clase gap-3 agrega separación entre los botones.
1. Botones apilados verticalmente (siempre)
2. Botones apilados en móvil, en fila en pantallas medianas o mayores
3. Botones centrados a partir de md
4. Botones alineados a la derecha a partir de md
