Tipografía: Código y Bloques de Código en Bootstrap 5

El elemento <code> se utiliza para resaltar fragmentos de código en línea dentro del texto.
Bootstrap 5 aplica estilos especiales al elemento <code> para hacerlo visualmente distintivo.
Ejemplo de código en línea:
Nuestra primera línea en Basic puede haber sido: 10 print "Hola Mundo"
<p>Nuestra primera línea en Basic puede haber sido <code>10 print "Hola Mundo"</code></p>
El elemento <pre> permite mostrar bloques de código con formato predefinido (espacios, saltos de línea, etc.).
Bootstrap también le aplica estilos para mejorar su legibilidad.
Ejemplo de bloque de código:
10 print "hola mundo"
20 goto 10
<pre>
  10 print "hola mundo"
  20 goto 10
</pre>
¿Cuándo usar cada uno?
  • <code>: Para resaltar una pequeña porción de código dentro de un párrafo.
  • <pre>: Para mostrar líneas completas o estructuras de código con formato original.
Accesibilidad: Usar correctamente estos elementos mejora la accesibilidad y comprensión del código.
