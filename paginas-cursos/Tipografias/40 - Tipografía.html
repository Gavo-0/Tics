Tamaños de títulos con etiquetas h1 a h6

¿Cómo funciona el tamaño de fuente en Bootstrap 5?
Bootstrap 5 utiliza un tamaño base de fuente de 16px (1rem) definido en la etiqueta <html>. Los encabezados h1-h6 usan valores relativos (rem y vw) para adaptarse tanto al tamaño base como al ancho de la pantalla, mejorando la legibilidad en diferentes dispositivos.
Etiqueta/Clase Tamaño en rem Tamaño en px
(en pantallas ≥1200px)

h2, .h22rem32px
h3, .h31.75rem28px
h4, .h41.5rem24px
h5, .h51.25rem20px
h6, .h61rem16px

Los encabezados HTML (h1 a h6) deben usarse para marcar títulos reales de secciones, ya que son importantes para la accesibilidad y el SEO. Si solo necesitas el estilo visual, pero no la semántica de título, puedes usar las clases .h1 a .h6 en cualquier etiqueta.

Tamaños de títulos con clases .h1 a .h6

¿Cuándo usar .h1 en lugar de <h1>?
Usa la clase .h1 cuando necesites un texto grande con el mismo tamaño de fuente que un encabezado, pero no sea realmente un título de sección. Ideal para resaltar palabras o frases dentro de un párrafo, botones, tarjetas u otros componentes.

Clases display-1 a display-6

¿Para qué sirven las clases display-1 a display-6?
Bootstrap ofrece las clases display-1 a display-6 para títulos más destacados, como los de portada, secciones principales o llamados de atención. Estas clases aplican un tamaño de fuente más grande y un peso ligero (font-weight: 300) para lograr un efecto moderno y llamativo.
Puedes usarlas en cualquier elemento, no solo en encabezados.

Etiqueta <small> y clase .small dentro de títulos

¿Qué hace la etiqueta <small>?
La etiqueta <small> y la clase .small reducen el tamaño del texto a 0.875em relativo al elemento padre. Perfecto para acotaciones, subtítulos o aclaraciones. Puedes combinarla con text-muted para tener un efecto visual más tenue.

Etiqueta <small class="text-muted"> para acotaciones tenues

Ejemplo práctico: Si quieres mostrar una descripción secundaria o información menos relevante, puedes usar <small class="text-muted"> para que el texto se vea más claro y discreto que el título principal.

Resumen:
  • Utiliza h1-h6 para títulos semánticos. Usa las clases h1-h6 para texto de igual tamaño pero sin semántica de título.
  • Las clases display-1 a display-6 permiten destacar títulos con tamaños aún mayores y modernos.
  • La etiqueta <small> o la clase .small reduce el tamaño de texto dentro de un título.
  • Agrega text-muted a <small> para acotaciones visualmente más tenues.
  • Todos los tamaños de fuente se calculan respecto al tamaño base de 16px (1rem) definido en html, lo que garantiza un diseño adaptable y accesible.
  • Usar la jerarquía de títulos correctamente mejora la accesibilidad y el SEO de tu página.
