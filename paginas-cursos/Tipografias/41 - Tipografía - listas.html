Curso de Bootstrap - Tipografía: Listas

Listas en Bootstrap 5

Bootstrap 5 provee estilos para listas ordenadas (<ol>), no ordenadas (<ul>), listas sin viñetas, listas en línea y listas de definición (<dl>), mejorando su presentación y adaptabilidad en cualquier dispositivo.

Listas ordenadas (<ol>)

Ejemplo de lista ordenada:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  4. Cuarto elemento
<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
  <li>Cuarto elemento</li>
</ol>

Listas no ordenadas (<ul>)

Ejemplo de lista no ordenada:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento
  • Cuarto elemento
<ul>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
  <li>Cuarto elemento</li>
</ul>

Listas sin viñetas y en línea

.list-unstyled elimina las viñetas y el margen izquierdo:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento
  • Cuarto elemento
<ul class="list-unstyled">
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
  <li>Cuarto elemento</li>
</ul>

.list-inline y .list-inline-item muestran la lista horizontalmente:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento
  • Cuarto elemento
<ul class="list-inline">
  <li class="list-inline-item">Primer elemento</li>
  <li class="list-inline-item">Segundo elemento</li>
  <li class="list-inline-item">Tercer elemento</li>
  <li class="list-inline-item">Cuarto elemento</li>
</ul>

Listas de definición (<dl>)

Las listas de definición son ideales para glosarios, descripciones o pares término-definición. Usan <dt> para el término y <dd> para la definición.
Palabra 1
Definición de la palabra 1
Palabra 2
Definición de la palabra 2
Palabra 3
Definición de la palabra 3
<dl>
  <dt>Palabra 1</dt>
  <dd>Definición de la palabra 1</dd>
  <dt>Palabra 2</dt>
  <dd>Definición de la palabra 2</dd>
  <dt>Palabra 3</dt>
  <dd>Definición de la palabra 3</dd>
</dl>

Lista de definición horizontal

Agrega la clase .row a <dl> y usa col-* en <dt> y <dd> para una presentación en filas.
Palabra 1
Definición de la palabra 1
Palabra 2
Definición de la palabra 2
Palabra 3
Definición de la palabra 3
<dl class="row">
  <dt class="col-sm-3">Palabra 1</dt>
  <dd class="col-sm-9">Definición de la palabra 1</dd>
  <dt class="col-sm-3">Palabra 2</dt>
  <dd class="col-sm-9">Definición de la palabra 2</dd>
  <dt class="col-sm-3">Palabra 3</dt>
  <dd class="col-sm-9">Definición de la palabra 3</dd>
</dl>

Resumen de clases útiles para listas en Bootstrap 5

Clase Descripción
list-unstyled Quita viñetas y margen izquierdo de la lista.
list-inline Muestra los elementos en línea horizontalmente.
list-inline-item Convierte el elemento <li> en item en línea dentro de .list-inline.
row (en dl) Utiliza el sistema de grillas para listas de definición horizontales.
col-* Define el ancho de cada término y definición en una lista de definición horizontal.
Consejos de buenas prácticas:
  • Utiliza listas ordenadas para pasos secuenciales o rankings, y no ordenadas para listados generales.
  • Las listas en línea son ideales para menús, tags o navegación secundaria.
  • Las listas de definición son perfectas para glosarios, términos técnicos o fichas de datos.
  • Combina las utilidades de Bootstrap con utilidades personalizadas para lograr la presentación deseada.
