Abreviaturas y Acrónimos en Bootstrap 5

El elemento <abbr> sirve para marcar abreviaturas y acrónimos en el HTML.
Cuando pasas el mouse sobre una abreviatura, se muestra el significado completo en un tooltip gracias al atributo title.
Bootstrap 5 mejora la apariencia y legibilidad de <abbr> y añade la clase .initialism para acrónimos (aparecen en mayúsculas y ligeramente más pequeños).
Ejemplo de abreviatura:
Este es un ejemplo de una abreviatura: Sr./a.
<abbr title="Señor/a">Sr./a.</abbr>
Ejemplo de acrónimo:
Este es un ejemplo de un acrónimo: ovni
<abbr title="Objeto Volador No Identificado" class="initialism">ovni</abbr>
¿Cuándo usar cada uno?
  • Abreviatura: Cuando una palabra se acorta (ej: "Sr./a." para Señor/a, "etc." para etcétera).
  • Acrónimo: Cuando una palabra se forma con las iniciales de varias palabras (ej: "ovni" para Objeto Volador No Identificado). Se recomienda usar la clase .initialism para resaltarlo.
Accesibilidad: El uso correcto de <abbr> mejora la accesibilidad y ayuda a lectores de pantalla a leer el significado completo.
