Curso de Bootstrap - Alineación horizontal

Alineación horizontal de columnas en Bootstrap 5

En Bootstrap 5 puedes alinear horizontalmente las columnas dentro de una fila utilizando utilidades de Flexbox. Esto es posible cuando la suma de anchos de columna es menor a 12, ya que si ocupan todo el espacio disponible, no se pueden mover horizontalmente.

Las utilidades más usadas para esto son las clases justify-content-* aplicadas sobre el .row. Estas clases definen cómo se distribuyen las columnas dentro del espacio restante.

Ejemplo de alineación horizontal

<div class="container">
  <div class="row justify-content-end">
    <div class="col-3">Columna 1</div>
    <div class="col-3">Columna 2</div>
  </div>

  <div class="row justify-content-start">
    <div class="col-3">Columna 1</div>
    <div class="col-3">Columna 2</div>
  </div>

  <div class="row justify-content-center">
    <div class="col-3">Columna 1</div>
    <div class="col-3">Columna 2</div>
  </div>

  <div class="row justify-content-between">
    <div class="col-3">Columna 1</div>
    <div class="col-3">Columna 2</div>
  </div>

  <div class="row justify-content-around">
    <div class="col-3">Columna 1</div>
    <div class="col-3">Columna 2</div>
  </div>

  <div class="row justify-content-evenly">
    <div class="col-3">Columna 1</div>
    <div class="col-3">Columna 2</div>
  </div>
</div>

Clases de alineación horizontal disponibles

  • justify-content-start: Alinea al inicio (izquierda en LTR, por defecto).
  • justify-content-end: Alinea al final (derecha en LTR).
  • justify-content-center: Centra las columnas horizontalmente.
  • justify-content-between: Deja máximo espacio entre columnas.
  • justify-content-around: Espacio igual a los lados y entre columnas.
  • justify-content-evenly: Espacio igual en todos los extremos y entre columnas.

Importante: Si la suma de las columnas es exactamente 12, no hay espacio para ver el efecto de alineación. Usa col-3 o col-4 para dejar espacio disponible.

Consejos prácticos y errores comunes

  • ✔️ Usa columnas que sumen menos de 12 para ver la alineación.
  • ❌ No uses márgenes manuales para forzar alineación, mejor usa las clases de Bootstrap.
  • ✔️ Prueba diferentes combinaciones con el inspector (F12) del navegador.
  • ✔️ Usa clases como .bg-light o .border para ayudarte a visualizar el espacio.
  • ❌ No uses justify-content en columnas directamente, solo en filas (.row).

¿Y si quiero alinear elementos dentro de una columna?

También puedes usar .d-flex dentro de una columna o cualquier otro elemento para aplicar alineaciones internas. Ejemplo: 

<div class="col d-flex justify-content-center align-items-center" style="height: 150px;">
  <p>Texto centrado horizontal y verticalmente</p>
</div>

Texto centrado horizontal y verticalmente

Esto es útil para botones, textos o íconos dentro de una tarjeta o sección.

Resumen

  • Utiliza justify-content-* en la fila para alinear horizontalmente.
  • La suma de las columnas debe ser menor a 12 para que haya espacio libre.
  • Flexbox permite alineación eficiente, sin necesidad de márgenes manuales.
  • También puedes usar d-flex dentro de columnas para control más preciso.
