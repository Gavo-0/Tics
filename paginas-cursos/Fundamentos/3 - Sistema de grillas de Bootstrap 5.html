Sistema de grillas de Bootstrap 5

El sistema de grillas (grid system) de Bootstrap es la base para crear diseños responsivos y flexibles. Divide el contenido en filas (row) y columnas (col-*), y cada fila puede tener hasta 12 unidades de columna.

Bootstrap se encarga de colapsar las columnas automáticamente en dispositivos con menor ancho, logrando que tu sitio sea amigable para computadoras, tablets y celulares.

Ejemplo de grid en Bootstrap 5

<div class="container">
  <div class="row">
    <div class="col-xxl-4" style="background-color:#aaa">
      <h1>Columna 1</h1>
      <p>Esto es una prueba de bootstrap.</p>
    </div>
    <div class="col-xxl-4" style="background-color:#bbb">
      <h1>Columna 2</h1>
      <p>Esto es una prueba de bootstrap.</p>
    </div>
    <div class="col-xxl-4" style="background-color:#ccc">
      <h1>Columna 3</h1>
      <p>Esto es una prueba de bootstrap.</p>
    </div>
  </div>
</div>

¿Cómo funciona? El grid utiliza clases como col-xxl-4, col-xl-4, col-lg-4, col-md-4, col-sm-4, col-4 para definir cuántas columnas ocupa cada elemento en distintos tamaños de pantalla.

  • col-xxl-*: colapsa cuando el ancho es menor a 1400px.
  • col-xl-*: colapsa cuando el ancho es menor a 1200px.
  • col-lg-*: colapsa cuando el ancho es menor a 992px.
  • col-md-*: colapsa cuando el ancho es menor a 768px.
  • col-sm-*: colapsa cuando el ancho es menor a 576px.
  • col-*: nunca colapsa.

Visualización del grid según el tamaño de pantalla

    • Columna 1
    Columna 2
    Columna 3

    • Columna 1
    Columna 2
    Columna 3
  • Ancho menor a 1200px: colapsan las columnas col-xl-4.
    • Columna 1
    Columna 2
    Columna 3
  • Ancho menor a 992px: colapsan las columnas col-lg-4.
    • Columna 1
    Columna 2
    Columna 3
  • Ancho menor a 768px: colapsan las columnas col-md-4, se apilan aún más.
    • Columna 1
    Columna 2
    Columna 3
  • Ancho menor a 576px: colapsan las columnas col-sm-4, normalmente se muestra una sola columna por fila.
    • Columna 1
    Columna 2
    Columna 3
  • Las columnas col-4 nunca colapsan: siempre mantienen el mismo ancho si hay espacio suficiente.
    • Columna 1
    Columna 2
    Columna 3

Significado de las clases de columnas

col-*        (todos los tamaños)
col-sm-*     (sm)all
col-md-*     (m)e(d)ium
col-lg-*     (l)ar(g)e
col-xl-*     (e)xtra (l)arge
col-xxl-*    (e)xtra (e)xtra (l)arge
Recomendación: Usa la combinación de clases de columna que mejor se adapte al comportamiento que buscas en tu página para que el diseño sea siempre amigable en todos los dispositivos.
