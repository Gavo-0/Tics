Anidamiento de columnas en Bootstrap 5

¿Qué es el anidamiento de columnas?

El anidamiento de columnas es una técnica que permite incluir una fila y columnas adicionales dentro de una columna existente. Esto permite construir diseños más sofisticados sin salirse del sistema de grillas de Bootstrap.

Ejemplo básico de anidamiento

En este ejemplo, tenemos una fila principal con dos columnas. Dentro de la segunda columna, anidamos una fila con otras dos columnas internas: 

<div class="container">
  <div class="row">
    <div class="col-lg-6" style="background:#aaa">
      <h1>Columna 1</h1>
    </div>
    <div class="col-lg-6" style="background:#bbb">
      <h1>Columna 2</h1>
      <div class="row">
        <div class="col-lg-6" style="background:#ccc">
          <p>Columna 2a</p>
        </div>
        <div class="col-lg-6" style="background:#ddd">
          <p>Columna 2b</p>
        </div>
      </div>
    </div>
  </div>
</div>

Como ves, la columna 2 contiene internamente dos columnas más. Bootstrap adapta todo automáticamente para que sea responsivo.

Ejemplo práctico: tarjeta con subcolumnas

Este patrón es útil para interfaces con información agrupada:

<div class="container">
  <div class="row">
    <div class="col-md-6">
      <div class="border p-3 bg-light rounded">
        <h5>Perfil de Usuario</h5>
        <div class="row">
          <div class="col-6">Nombre: Ana</div>
          <div class="col-6">Edad: 28</div>
        </div>
      </div>
    </div>
  </div>
</div>

Recomendaciones importantes

  • Asegúrate de usar .row dentro de una columna, no directamente dentro de .container.
  • La suma de columnas anidadas también debe ser 12 o menos.
  • Evita anidar demasiadas veces para no complicar el mantenimiento del código.

Resumen

  • El anidamiento permite crear diseños complejos manteniendo la estructura responsiva.
  • Se puede usar .row y .col-* dentro de cualquier .col.
  • El sistema de grillas de Bootstrap mantiene la alineación y distribución de forma automática.
