Curso de Bootstrap - Offset en columnas

Uso de Offset en Bootstrap 5

Podemos dejar columnas vacías indicando un desplazamiento en el momento de crearla.

Problema

Implementar el código con Bootstrap 5 necesario para mostrar los números "uno", "dos", etc. con el siguiente formato:

Uno

Dos

Tres

Cuatro

Cinco

<div class="container">
  <div class="row">
    <div class="col-lg-4" style="background-color:#aaa"><h1>Uno</h1></div>
    <div class="col-lg-4 offset-lg-4" style="background-color:#bbb"><h1>Dos</h1></div>
  </div>

  <div class="row">
    <div class="col-lg-4 offset-lg-4" style="background-color:#aaa"><h1>Tres</h1></div>
  </div>

  <div class="row">
    <div class="col-lg-4" style="background-color:#aaa"><h1>Cuatro</h1></div>
    <div class="col-lg-4 offset-lg-4" style="background-color:#bbb"><h1>Cinco</h1></div>
  </div>
</div>

Las clases disponibles para desplazamiento horizontal son:

offset-*, offset-sm-*, offset-md-*, offset-lg-*, offset-xl-*, offset-xxl-*

El * representa un valor entre 1 y 12. Esto permite espaciar columnas horizontalmente sin necesidad de usar columnas vacías.
