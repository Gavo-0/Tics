Curso de Bootstrap - Múltiples reglas en columnas

Múltiples reglas a las columnas de una fila

El sistema de grillas de Bootstrap 5 permite asignar diferentes reglas de ancho a las columnas de una fila, según el tamaño de pantalla del dispositivo. Así, puedes lograr que el diseño se adapte perfectamente a computadoras, tablets y celulares, sin necesidad de escribir CSS adicional.

Esto se logra aplicando varias clases col-* en un mismo div. Bootstrap elegirá la regla más específica válida para el ancho del dispositivo.

Breakpoints disponibles en Bootstrap 5

Breakpoint Clase Ancho mínimo
Extra pequeño col- 0 px
Pequeño col-sm-576 px
Mediano col-md-768 px
Grande col-lg-992 px
Extra grande col-xl-1200 px
Extra-extra grandecol-xxl-1400 px

Ejemplo práctico: dos columnas con anchos adaptativos

Puedes definir diferentes anchos para cada columna según el tamaño de pantalla del dispositivo: 


1er columna              2da columna

Pantalla muy,muy grande:      10 unidades              2 unidades
Pantalla muy grande:           9 unidades              3 unidades
Pantalla grande:               8 unidades              4 unidades
Pantalla mediana:              7 unidades              5 unidades
Pantalla chica:                6 unidades              6 unidades
Pantalla muy chica:            5 unidades              7 unidades

El código HTML quedaría así: 

<div class="container">
  <div class="row">
    <div class="col-xxl-10 col-xl-9 col-lg-8 col-md-7 col-sm-6 col-5" style="background-color:#aaa">
      <h1>Columna 1</h1>
      <p>Esto es una prueba de bootstrap 5.</p>
    </div>
    <div class="col-xxl-2 col-xl-3 col-lg-4 col-md-5 col-sm-6 col-7" style="background-color:#bbb">
      <h1>Columna 2</h1>
      <p>Esto es una prueba de bootstrap 5.</p>
    </div>
  </div>
</div>

¿Qué sucede al cambiar el tamaño de la pantalla?

  • Si el ancho es ≥1400px, se aplican col-xxl-10 y col-xxl-2.
  • Si es ≥1200px y <1400px, se aplican col-xl-9 y col-xl-3.
  • Si es ≥992px y <1200px, se aplican col-lg-8 y col-lg-4.
  • Si es ≥768px y <992px, se aplican col-md-7 y col-md-5.
  • Si es ≥576px y <768px, se aplican col-sm-6 y col-sm-6.
  • Si es <576px, se aplican col-5 y col-7.

Bootstrap toma la clase más adecuada según el ancho de pantalla, manteniendo siempre una suma de 12 unidades por fila.

Ventajas de usar múltiples reglas de columnas

  • Diseño responsivo sin necesidad de CSS personalizado.
  • Control total sobre el layout para cada tamaño de dispositivo.
  • Fácil de mantener y expandir.
  • Compatible con otros componentes de Bootstrap.

Buenas prácticas

  • La suma de columnas por fila debe ser 12 para mantener el orden y evitar columnas vacías.
  • Usa siempre .container o .container-fluid para centrar y limitar el ancho.
  • Agrega padding (p-3) o gap (g-2) para separar columnas.
  • Prueba en múltiples dispositivos o usa las herramientas del navegador.

La definición de múltiples reglas es una de las herramientas más poderosas de Bootstrap, permitiéndote definir distintos diseños de página según las capacidades del dispositivo del visitante.
