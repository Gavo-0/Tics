Tarjeta con Pestañas Bootstrap 5

Añade pestañas interactivas en la cabecera de una tarjeta con Bootstrap 5 para separar contenido o flujos de trabajo.
  • Haz clic en cada pestaña para mostrar su contenido correspondiente.
  • Puedes colocar formularios, listas, alertas, tablas, imágenes, etc., en cada pestaña.
  • Incluye ejemplos avanzados y buenas prácticas para tabulación accesible.
  • Incluye ejemplo con píldoras y ejemplo con contenido dinámico (JS).

Código de ejemplo básico

<div class="card">
  <div class="card-header">
    <ul class="nav nav-tabs card-header-tabs">
      <li class="nav-item">
        <button class="nav-link active" data-bs-toggle="tab" data-bs-target="#uno">Uno</button>
      </li>
      <li class="nav-item">
        <button class="nav-link" data-bs-toggle="tab" data-bs-target="#dos">Dos</button>
      </li>
    </ul>
  </div>
  <div class="card-body">
    <div class="tab-content">
      <div class="tab-pane fade show active" id="uno">Contenido uno</div>
      <div class="tab-pane fade" id="dos">Contenido dos</div>
    </div>
  </div>
</div>

Código de ejemplo con píldoras

<ul class="nav nav-pills card-header-pills">
  <li class="nav-item">
    <button class="nav-link active" data-bs-toggle="pill" data-bs-target="#pillA">Píldora A</button>
  </li>
  <li class="nav-item">
    <button class="nav-link" data-bs-toggle="pill" data-bs-target="#pillB">Píldora B</button>
  </li>
</ul>
<div class="tab-content">
  <div class="tab-pane fade show active" id="pillA">Contenido A</div>
  <div class="tab-pane fade" id="pillB">Contenido B</div>
</div>
