Alertas con Bootstrap 5

Bootstrap 5 permite mostrar mensajes destacados a los usuarios de manera simple, visual y profesional, usando colores y estilos fácilmente reconocibles.

¿Para qué sirven las alertas?
Las alertas se usan para mostrar advertencias, errores, confirmaciones o información importante. Cada color transmite un significado:
  • Azul (primary): Información general
  • Verde (success): Éxito
  • Rojo (danger): Error o peligro
  • Amarillo (warning): Advertencia
  • Gris (secondary, light, dark): Neutro o contexto especial
  • Celeste (info): Información secundaria

¿Cómo se crea una alerta?

Estructura básica:
Usa un <div> con la clase base alert y una clase de color contextual:
<div class="alert alert-success">
  ¡Operación realizada con éxito!
</div>
¡Operación realizada con éxito!
Clases de color disponibles:
alert-primary, alert-secondary, alert-success, alert-danger, alert-warning, alert-info, alert-light, alert-dark

Ejemplo visual: Todos los colores de alerta

Combina las alertas en una grilla para que el usuario vea rápidamente todas las opciones:
Primary: Mensaje informativo.
Secondary: Mensaje neutro.
Success: ¡Acción exitosa!
Danger: ¡Ha ocurrido un error!
Warning: Atención, revisa esto.
Info: Dato adicional para el usuario.
Light: Fondo claro, poco intrusivo.
Dark: Fondo oscuro, contraste fuerte.

Alertas enriquecidas (título, enlace y separador)

Puedes agregar títulos, enlaces y divisores para que la alerta sea más clara y útil:

¡Atención!

El navegador más utilizado es Chrome.

Si tienes dudas, consulta al administrador.

<div class="alert alert-info">
  <h4 class="alert-heading">¡Atención!</h4>
  <p>
    El navegador más utilizado es 
    <a class="alert-link" href="https://www.google.com/chrome/">Chrome</a>.
  </p>
  <hr>
  <p class="mb-0">Si tienes dudas, consulta al administrador.</p>
</div>

¿Cómo activar funcionalidades interactivas?

Si quieres que las alertas puedan cerrarse (descartarse), debes importar el JavaScript de Bootstrap al final del <body>: 
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0/dist/js/bootstrap.bundle.min.js" crossorigin="anonymous"></script>

Alertas cerrables (con botón de cierre)

  • Agrega la clase alert-dismissible a la alerta.
  • Agrega el botón con btn-close y data-bs-dismiss="alert".
  • Incluye fade show para animar el cierre.
¡Error! No se pudo guardar tu información. 
<div class="alert alert-danger alert-dismissible fade show" role="alert">
  <strong>¡Error!</strong> No se pudo guardar tu información.
  <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>
<!-- Recuerda importar el JS de Bootstrap al final del body -->

Resumen rápido
