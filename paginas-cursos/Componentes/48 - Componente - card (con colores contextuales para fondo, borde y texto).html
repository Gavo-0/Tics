Tarjetas con Colores Contextuales

Bootstrap 5 provee una variedad de clases contextuales para cambiar el color de fondo, borde y texto de las tarjetas (cards). Esto permite destacar información importante y mejorar la presentación visual.

Colores de fondo para tarjetas

Usa las clases bg-primary, bg-success, bg-info, bg-warning, bg-danger, bg-secondary, bg-dark y bg-light para fijar el color de fondo de una tarjeta. Añade text-white para que el texto sea legible sobre fondos oscuros. 
<div class="card bg-success text-white">...</div>
<div class="card bg-warning text-white">...</div>
<div class="card bg-danger text-white">...</div>

Colores de borde para tarjetas

Puedes resaltar una tarjeta usando las clases de borde: border-primary, border-success, border-info, border-warning, border-danger, border-secondary, border-dark y border-light. 
<div class="card border-success">...</div>
<div class="card border-warning">...</div>
<div class="card border-danger">...</div>
