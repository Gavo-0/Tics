Tamaños y posiciones del Modal en Bootstrap 5

Puedes modificar el tamaño del cuadro de diálogo (modal) usando:
modal-sm, modal-lg, modal-xl, modal-fullscreen
También puedes centrarlo con modal-dialog-centered o hacerlo fullscreen según el dispositivo.

Ejemplo: Modal pequeño y centrado

Modal pequeño

Este es un modal pequeño y centrado.

</div>

Ejemplo: Modal extra grande (modal-xl)

Modal extra grande

Este es un modal de tamaño extra grande.

Ejemplo: Modal fullscreen

Modal a pantalla completa

Este es un modal que ocupa toda la pantalla.
Útil para dispositivos móviles o para mostrar mucho contenido. 
<div class="modal-dialog modal-fullscreen">
  ...
</div>

Modal fullscreen según dispositivo

Para mostrar el modal en pantalla completa solo en ciertos tamaños usa:
modal-fullscreen-sm-down, modal-fullscreen-md-down, modal-fullscreen-lg-down, modal-fullscreen-xl-down, modal-fullscreen-xxl-down 
<div class="modal-dialog modal-fullscreen-md-down">
  ...
</div>
