Cuadros de diálogo (Modal) en Bootstrap 5

Los modales permiten mostrar ventanas emergentes superpuestas sobre el contenido principal.
Úsalos para mostrar información importante, formularios o confirmaciones.

¡Importante!
Los modales de Bootstrap requieren la librería JavaScript de Bootstrap. No olvides importarla al final del <body>.

Estructura básica de un modal

  • El contenedor principal debe tener clases modal y fade, además de un id único.
  • Dentro hay un modal-dialog y un modal-content.
  • El contenido se divide en: modal-header (cabecera), modal-body (cuerpo) y modal-footer (pie).
Título del Modal
¡Hola! Soy un diálogo modal de Bootstrap 5.
Puedes cerrarme con la X, el botón de abajo o haciendo clic fuera del cuadro.
Solo puede haber un modal abierto a la vez. 
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#dialogo1">Abrir ventana de diálogo</button>

<div class="modal fade" id="dialogo1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Título del diálogo</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        Contenido del diálogo.
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Cerrar</button>
      </div>
    </div>
  </div>
</div>
<!-- Al final del body agrega: -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0/dist/js/bootstrap.bundle.min.js"></script>

Evitar que se cierre el modal al hacer clic fuera

Para que el usuario solo pueda cerrar el modal con los botones, añade data-bs-backdrop="static" en el div principal: 
<div class="modal fade" id="dialogo1" data-bs-backdrop="static">
  ...
</div>

Modal con contenido largo (scroll interno)

Si el contenido es extenso, añade la clase modal-dialog-scrollable al div modal-dialog para mostrar una barra de desplazamiento dentro del modal: 
<div class="modal fade" id="dialogo2">
  <div class="modal-dialog modal-dialog-scrollable">
    ...
  </div>
</div>

Importar JavaScript de Bootstrap

¡No olvides este paso!
Agrega la siguiente línea al final del <body> para que los modales funcionen: 
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0/dist/js/bootstrap.bundle.min.js"></script>
