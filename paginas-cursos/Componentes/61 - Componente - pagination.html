Paginadores en Bootstrap 5: Clases y ejemplos

Para mostrar un paginador de datos en Bootstrap 5, utiliza una lista <ul> con la clase pagination. Cada ítem de página lleva la clase page-item y cada enlace la clase page-link.

  • pagination: Se aplica al elemento ul principal.
  • page-item: Se aplica a cada li que representa una página.
  • page-link: Se aplica a cada a dentro de los li.

Ejemplo básico de paginador

<ul class="pagination">
  <li class="page-item"><a class="page-link" href="#"><</a></li>
  <li class="page-item"><a class="page-link" href="#">1</a></li>
  <li class="page-item"><a class="page-link" href="#">2</a></li>
  <li class="page-item"><a class="page-link" href="#">3</a></li>
  <li class="page-item"><a class="page-link" href="#">4</a></li>
  <li class="page-item"><a class="page-link" href="#">5</a></li>
  <li class="page-item"><a class="page-link" href="#">6</a></li>
  <li class="page-item"><a class="page-link" href="#">></a></li>
</ul>

Tamaños de paginación

Puedes mostrar los enlaces de paginación en tamaño pequeño (pagination-sm) o grande (pagination-lg). 

<ul class="pagination pagination-sm"> ... </ul>
<ul class="pagination pagination-lg"> ... </ul>

Página activa

Para indicar la página activa, agrega la clase active al elemento li correspondiente. 

<li class="page-item active"><a class="page-link" href="#">2</a></li>

Alineación de la paginación

Puedes alinear el paginador usando las clases justify-content-center, justify-content-start o justify-content-end en el elemento ul. 

<ul class="pagination justify-content-center"> ... </ul>
<ul class="pagination justify-content-end"> ... </ul>
