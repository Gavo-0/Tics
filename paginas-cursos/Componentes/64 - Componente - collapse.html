Collapse: Mostrar y ocultar contenido en Bootstrap 5

La componente collapse permite mostrar u ocultar contenido de forma dinámica, utilizando botones o enlaces. Es útil para mostrar información adicional sin recargar la interfaz.

  • collapse: Clase que aplica el comportamiento de mostrar/ocultar.
  • Se activa mediante data-bs-toggle="collapse" y un href o data-bs-target apuntando al id del contenido.

Collapse con hipervínculo

<a href="#politica" data-bs-toggle="collapse">Políticas de privacidad</a>
<div id="politica" class="collapse">...</div>

Collapse visible al cargar la página

Agrega la clase show para que el contenido aparezca visible al cargar. 

<div id="politica" class="collapse show">...</div>

Collapse con botón

<button class="btn btn-warning" data-bs-toggle="collapse" data-bs-target="#politica3">Políticas de privacidad</button>
<div id="politica3" class="collapse">...</div>

Collapse de múltiples áreas

Se pueden mostrar y ocultar varias áreas independientes usando data-bs-target y diferentes id. 

<a href="#area1" data-bs-toggle="collapse" data-bs-target="#area1">Área 1</a>
<a href="#area2" data-bs-toggle="collapse" data-bs-target="#area2">Área 2</a>

<div id="area1" class="collapse multi-collapse"><div class="card card-body">...</div></div>
<div id="area2" class="collapse multi-collapse"><div class="card card-body">...</div></div>
