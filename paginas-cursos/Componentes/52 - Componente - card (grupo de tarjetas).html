Curso de Bootstrap - Tarjetas (Cards)

Agrupando tarjetas con card-group en Bootstrap 5

Bootstrap permite agrupar tarjetas para que tengan el mismo ancho y altura usando la clase card-group.

¿Cómo funciona?
Al envolver varias tarjetas (.card) dentro de un div con clase card-group, todas tendrán el mismo alto y se verán alineadas. Perfecto para mostrar información uniforme, como productos, servicios o características.

¿Cuándo usar card-group?
  • Cuando necesitas mostrar varias tarjetas alineadas horizontalmente con la misma altura.
  • Ideal para galerías, productos, características, etc.
  • En dispositivos móviles, las tarjetas se apilan verticalmente.
Tip: Para agregar separación entre tarjetas, usa la grilla de Bootstrap (row + col) y utilidades g-*.
