Tarjetas (Cards) en Bootstrap 5

En Bootstrap 5, una tarjeta (“card”) es un componente flexible y extensible que sirve como contenedor de información. Las tarjetas permiten organizar el contenido de forma clara, visual y atractiva, facilitando la presentación de información relacionada, imágenes, enlaces y más, dentro de bloques estilizados con bordes, sombras, fondos y distintas opciones de personalización.

¿Qué es una tarjeta (card)?

Una tarjeta es un bloque visual que agrupa información y elementos relacionados, como títulos, textos, imágenes, enlaces, listas y botones. Las tarjetas pueden incluir encabezados, pies de página, imágenes de cabecera o fondo, colores de fondo contextuales y se pueden apilar, alinear en filas o columnas, o combinar con otros componentes Bootstrap.
  • Encabezado: Puede usarse para mostrar el título, subtítulo o icono.
  • Cuerpo: Es el área principal para textos, imágenes u otros elementos.
  • Pie de página: Permite añadir información adicional o acciones.
  • Colores y fondos: Se pueden utilizar las clases contextuales como .bg-primary, .bg-success, etc.
  • Botones y enlaces: Es común añadir botones de acción o enlaces dentro de la tarjeta.

Estructura básica de una tarjeta

<div class="card">
  <div class="card-body">
    <h4 class="card-title">Título de la tarjeta</h4>
    <p class="card-text">Contenido de la tarjeta.</p>
    <a href="#" class="btn btn-primary">Acción</a>
  </div>
</div>

Nota: Puedes centrar el contenido de la tarjeta añadiendo la clase text-center al div.card.

Ejemplo: Tres tarjetas iguales en la misma fila

A continuación, se muestra cómo crear tres tarjetas básicas, ubicadas en la misma fila y con el mismo ancho usando el sistema de grillas de Bootstrap. Cada tarjeta contiene un título, texto y un botón. 
<div class="container">
  <div class="row">
    <div class="col-md">
      <div class="card text-center">
        <div class="card-body">
          <h4 class="card-title">Título de la tarjeta 1</h4>
          <p class="card-text">Contenido de la tarjeta 1.</p>
          <a href="#" class="btn btn-primary">Entrar</a>
        </div>
      </div>
    </div>
    ... (más columnas con tarjetas)
  </div>
</div>

Personalización y variantes

  • Imágenes: Puedes agregar imágenes usando <img class="card-img-top"> para mostrar una imagen en la parte superior.
  • Colores de fondo y bordes: Utiliza clases como .bg-primary, .border-success para cambiar el color de fondo o el borde.
  • Listas dentro de tarjetas: Puedes usar <ul class="list-group list-group-flush"> para mostrar listas.
  • Grupos de tarjetas: Puedes agrupar varias tarjetas usando .card-group para mantenerlas del mismo tamaño y altura.
  • Sombras: Añade la clase .shadow o .shadow-sm para darle profundidad visual.
<div class="card mb-3">
  <img src="https://via.placeholder.com/500x200" class="card-img-top" alt="Imagen de ejemplo">
  <div class="card-body">
    <h5 class="card-title">Tarjeta con imagen</h5>
    <p class="card-text">Puedes añadir imágenes a tus tarjetas fácilmente.</p>
  </div>
</div>
