Bootstrap 5: Popover

Un Popover es una ventana emergente contextual que aparece tras hacer clic o enfocar sobre un elemento. A diferencia de un tooltip, el popover puede mostrar títulos, párrafos, listas, enlaces y formato HTML personalizado. Es ideal para mostrar información adicional, tips, ayuda, detalles o instrucciones sobre botones, enlaces, iconos, etc.
¿Cómo se usa?
  • Importa los archivos CSS y JavaScript de Bootstrap.
  • Agrega data-bs-toggle="popover" al elemento que mostrará el popover.
  • Define el contenido principal con data-bs-content y el título con title.
  • Personaliza la posición con data-bs-placement: top, bottom, left, right.
  • data-bs-trigger="focus" lo cierra automáticamente al hacer clic fuera.
  • Inicializa los popovers con JavaScript (ver ejemplo abajo).
<script>
  var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'))
  var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl)
  })
</script>

Ejemplos prácticos de Popover

Popover en botón clásico

<button type="button" class="btn btn-primary"
  data-bs-toggle="popover"
  title="¿Qué es Bootstrap?"
  data-bs-content="Bootstrap es un framework para crear sitios web modernos y responsivos."
  data-bs-placement="top"
  data-bs-trigger="focus">
  Saber más
</button>

Popover con formato HTML

<button class="btn btn-success"
  data-bs-toggle="popover"
  title="Atajo útil"
  data-bs-html="true"
  data-bs-content="Puedes usar <b>Ctrl+C</b> para copiar y <span style='color:#7952B3;'>Ctrl+V</span> para pegar."
  data-bs-placement="bottom"
  data-bs-trigger="focus">
  Ver atajo
</button>

Popover en icono

<i class="bi bi-question-circle" tabindex="0"
  data-bs-toggle="popover"
  title="Ayuda rápida"
  data-bs-content="Haz clic aquí si necesitas soporte o asistencia personalizada."
  data-bs-placement="right"
  data-bs-trigger="focus"></i>

Popover en listas y enlaces

  • Google
  • Yahoo
  • Bing
  • Texto interactivo con Popover 
<a href="#" data-bs-toggle="popover" title="Historia de Google" data-bs-trigger="focus"
  data-bs-content="Larry Page y Sergey Brin empezaron Google en 1996 en Stanford.">Google</a>

<span tabindex="0" data-bs-toggle="popover" data-bs-html="true" title="Popover en texto"
  data-bs-content="Puedes mostrar <b>popover</b> en cualquier <em>texto</em>.">
  Texto interactivo con Popover
</span>
TIP: Si no agregas data-bs-trigger="focus", el popover se cierra haciendo clic otra vez en el elemento. Puedes usar data-bs-html="true" para mostrar HTML como listas, imágenes, enlaces y más dentro del popover.
