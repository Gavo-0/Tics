Carousel (Carrusel) de imágenes en Bootstrap 5

Un carousel es un componente que permite mostrar un conjunto de diapositivas (imágenes o contenido) en secuencia, avanzando de forma automática o mediante controles de usuario.
Incluye:
  • Transición automática entre diapositivas.
  • Botones de navegación a izquierda y derecha.
  • Indicadores (puntos) para elegir la diapositiva.
  • Opcionalmente, título y descripción sobre cada imagen.
Recuerda: el carousel requiere tanto el CSS como el JavaScript de Bootstrap para funcionar correctamente.

Ejemplo para copiar y pegar

<div id="carousel1" class="carousel slide" data-bs-ride="carousel">
  <div class="carousel-indicators">
    <button type="button" data-bs-target="#carousel1" data-bs-slide-to="0" class="active" aria-current="true" aria-label="Diapositiva 1"></button>
    <button type="button" data-bs-target="#carousel1" data-bs-slide-to="1" aria-label="Diapositiva 2"></button>
    <button type="button" data-bs-target="#carousel1" data-bs-slide-to="2" aria-label="Diapositiva 3"></button>
  </div>
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img class="d-block w-100" src="https://www.tutorialesprogramacionya.com/imagenes/cuadro1.jpg" alt="Diapositiva 1">
      <div class="carousel-caption d-none d-md-block">
        <h3>Primera imagen</h3>
        <p>Descripción de la imagen 1.</p>
      </div>
    </div>
    <div class="carousel-item">
      <img class="d-block w-100" src="https://www.tutorialesprogramacionya.com/imagenes/cuadro2.jpg" alt="Diapositiva 2">
      <div class="carousel-caption d-none d-md-block">
        <h3>Segunda imagen</h3>
        <p>Descripción de la imagen 2.</p>
      </div>
    </div>
    <div class="carousel-item">
      <img class="d-block w-100" src="https://www.tutorialesprogramacionya.com/imagenes/cuadro3.jpg" alt="Diapositiva 3">
      <div class="carousel-caption d-none d-md-block">
        <h3>Tercera imagen</h3>
        <p>Descripción de la imagen 3.</p>
      </div>
    </div>
  </div>
  <button class="carousel-control-prev" type="button" data-bs-target="#carousel1" data-bs-slide="prev">
    <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Anterior</span>
  </button>
  <button class="carousel-control-next" type="button" data-bs-target="#carousel1" data-bs-slide="next">
    <span class="carousel-control-next-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Siguiente</span>
  </button>
</div>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0/dist/js/bootstrap.bundle.min.js"></script>
  • Asegúrate de que el id del carousel sea el mismo en los data-bs-target de los controles e indicadores.
  • El primer carousel-item debe tener la clase active.
  • Puedes agregar o quitar diapositivas según lo necesites.
  • Coloca el <script> de Bootstrap Bundle al final de tu archivo para que funcione el carousel.
