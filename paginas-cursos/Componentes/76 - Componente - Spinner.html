Spinners en Bootstrap 5: Indicadores visuales de carga

Los spinners son gráficos animados que indican el estado de carga o espera en una página o sección. Bootstrap 5 ofrece spinners circulares y spinners de crecimiento en todos los colores contextuales. No requieren JavaScript y son altamente personalizables.
¿Para qué sirven los spinners?
  • Mostrar que una acción está en proceso (carga de datos, envío de formularios, etc).
  • Indicar al usuario que debe esperar antes de interactuar con la interfaz.
  • Mejorar la experiencia visual en aplicaciones web y móviles.
Ventajas: Son ligeros, responsivos y pueden integrarse en cualquier elemento (botones, tarjetas, overlays).

Personalización adicional

  • Puedes modificar el tamaño de los spinners con las clases spinner-border-sm y spinner-grow-sm, o usando estilos CSS personalizados.
  • Usa role="status" y span.visually-hidden para accesibilidad.
  • Coloca el spinner dentro de cualquier contenedor, botón, tarjeta, overlay, etc.
  • Utiliza colores contextuales para integrarlo con el diseño de tu app.

¡Recuerda! Los spinners se usan para mejorar la experiencia visual y guiar al usuario durante cargas o procesos, evitando incertidumbre.
