Offcanvas en Bootstrap 5: Barras laterales ocultas

La componente Offcanvas permite mostrar una barra lateral (o panel) oculta que aparece sobre la interfaz cuando el usuario lo solicita (por ejemplo, al tocar un botón). Es ideal para navegación lateral, menús en móviles o paneles de opciones adicionales. El panel puede aparecer desde la izquierda, derecha, arriba o abajo según la clase utilizada.

¿Cómo funciona?
  • El panel se activa con un botón o enlace usando data-bs-toggle="offcanvas".
  • La dirección desde la que aparece se define con clases como offcanvas-start (izquierda), offcanvas-end (derecha), offcanvas-top (arriba), offcanvas-bottom (abajo).
  • Sólo se puede mostrar un Offcanvas a la vez (como los diálogos modal).
  • Incluye una cabecera y un área de cuerpo para tu contenido.

Ejemplo: Offcanvas lateral izquierdo (Login)

Abrir Login 
<a class="btn btn-primary" data-bs-toggle="offcanvas" href="#offcanvas1">Login</a>
<div class="offcanvas offcanvas-start" tabindex="-1" id="offcanvas1">
  <div class="offcanvas-header">
    <h5 class="offcanvas-title">Login</h5>
    <button type="button" class="btn-close text-reset" data-bs-dismiss="offcanvas"></button>
  </div>
  <div class="offcanvas-body">
    ...contenido...
  </div>
</div>

Ejemplo: Offcanvas desde otras posiciones

<button class="btn btn-success" data-bs-toggle="offcanvas" data-bs-target="#offcanvasRight">Desde la derecha</button>
<div class="offcanvas offcanvas-end" id="offcanvasRight">...</div>

<button class="btn btn-info" data-bs-toggle="offcanvas" data-bs-target="#offcanvasTop">Desde arriba</button>
<div class="offcanvas offcanvas-top" id="offcanvasTop">...</div>

<button class="btn btn-warning" data-bs-toggle="offcanvas" data-bs-target="#offcanvasBottom">Desde abajo</button>
<div class="offcanvas offcanvas-bottom" id="offcanvasBottom">...</div>
Panel derecho

Este panel aparece desde la derecha de la pantalla.

  • Ideal para menús de usuario
  • Configuración rápida
Panel superior

Este panel aparece desde arriba. Útil para notificaciones o accesos rápidos.

Panel inferior

Este panel aparece desde abajo de la pantalla.

¡Puede usarse para avisos importantes!

Resumen de posiciones Offcanvas

  • offcanvas-start: Aparece de izquierda a derecha.
  • offcanvas-end: Aparece de derecha a izquierda.
  • offcanvas-top: Aparece de arriba hacia abajo.
  • offcanvas-bottom: Aparece de abajo hacia arriba.

Nota: El contenido puede ser cualquier cosa: formularios, menús, alertas, listas, etc. Ideal para móviles o para mostrar información extra sin recargar la interfaz principal.
