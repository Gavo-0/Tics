Curso de Bootstrap - Nav-tabs y Nav-pills

Pestañas y contenido asociado en Bootstrap 5

Puedes asociar paneles de contenido a tu menú de pestañas (nav-tabs) o pastillas (nav-pills). Solo se muestra el contenido de la opción seleccionada. Bootstrap 5 gestiona el cambio de contenido automáticamente usando JavaScript.

  • nav-tabs y nav-pills: Menú horizontal de opciones.
  • tab-content: Contenedor de paneles de contenido.
  • tab-pane: Panel de cada pestaña/pastilla, vinculado por id.
  • data-bs-toggle="tab": Activa el comportamiento de pestaña.
  • show active: Inicializa la pestaña y el contenido seleccionados.
  • flex-column: Permite menú vertical.

Ejemplo: Pestañas con contenido

<ul class="nav nav-tabs mt-3">
  <li class="nav-item">
    <a class="nav-link active" data-bs-toggle="tab" href="#opcion1">Opción 1</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" data-bs-toggle="tab" href="#opcion2">Opción 2</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" data-bs-toggle="tab" href="#opcion3">Opción 3</a>
  </li>
</ul>
<div class="tab-content">
  <div class="tab-pane fade show active" id="opcion1">Contenido 1</div>
  <div class="tab-pane fade" id="opcion2">Contenido 2</div>
  <div class="tab-pane fade" id="opcion3">Contenido 3</div>
</div>

Detalles de implementación

  • Cada nav-link debe tener data-bs-toggle="tab" y un href que coincida con el id del panel de contenido.
  • El panel de contenido debe tener las clases tab-pane fade y sólo el activo debe tener además show active.
  • Puedes usar nav-tabs o nav-pills, y horizontal o vertical (flex-column).
  • Recuerda importar bootstrap.bundle.min.js para la funcionalidad dinámica.

Nota: Este sistema de pestañas y paneles es ideal para organizar grandes volúmenes de información en un espacio reducido, permitiendo una navegación dinámica y fluida.
