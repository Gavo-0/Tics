Bootstrap 5: Tooltips

Un Tooltip es un pequeño mensaje que aparece cuando el usuario pasa el cursor sobre un elemento. Bootstrap 5 permite mostrar tooltips en botones, enlaces, iconos y otros elementos HTML, personalizando su contenido, posición y formato fácilmente.

¿Cómo funcionan los tooltips?

  • Debes importar el CSS y el JavaScript de Bootstrap.
  • Agrega data-bs-toggle="mensaje" (o tooltip si prefieres el estándar) al elemento que tendrá el tooltip.
  • Define el texto o HTML del tooltip en el atributo title.
  • Opcional: Personaliza la posición con data-bs-placement (top, bottom, left, right).
  • Opcional: Para mensaje con formato HTML, agrega data-bs-html="true" y usa etiquetas en title.
  • Inicializa los tooltips con JavaScript:
<script>
  var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="mensaje"]'))
  var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
    return new bootstrap.Tooltip(tooltipTriggerEl)
  })
</script>

Ejemplo básico: Tooltip en botones de buscadores

Google Bing Yahoo 
<a data-bs-toggle="mensaje" title="Google Inc." href="https://www.google.com" class="btn btn-success">Google</a>
<a data-bs-toggle="mensaje" title="Microsoft" href="https://www.bing.com" class="btn btn-primary">Bing</a>
<a data-bs-toggle="mensaje" title="Yahoo Inc" href="https://www.yahoo.com" class="btn btn-warning text-dark">Yahoo</a>

Ejemplo avanzado: Posición y formato HTML en el mensaje

Tooltip derecha 
<button type="button" class="btn btn-secondary"
  data-bs-toggle="mensaje" data-bs-placement="bottom"
  title="Tooltip en la parte inferior">
  Tooltip abajo <i class="bi bi-arrow-down"></i>
</button>

<span tabindex="0" class="btn btn-outline-info"
  data-bs-toggle="mensaje" data-bs-placement="right"
  title="Tooltip a la derecha">
  Tooltip derecha <i class="bi bi-arrow-right"></i>
</span>

<button class="btn btn-danger"
  data-bs-toggle="mensaje" data-bs-html="true"
  data-bs-placement="top"
  title="Este es un <b style='color:yellow;'>tooltip</b> <em style='color:#fff;background:#7952B3;border-radius:4px;padding:2px 6px;'>con formato</em>">
  Tooltip con HTML <i class="bi bi-info-circle"></i>
</button>

Más ejemplos de Tooltips

Texto con tooltip 
<i class="bi bi-github" data-bs-toggle="mensaje" title="¡Síguenos en GitHub!"></i>
<img src="logo.svg" data-bs-toggle="mensaje" data-bs-placement="top" title="Logo Markdown">
<span data-bs-toggle="mensaje" data-bs-html="true"
  title="Pasa el <b>mouse</b> por aquí para ver este <span style='color:#7952B3;'>tooltip</span> con <em>HTML</em>">
  Texto con tooltip
</span>
TIP: Puedes usar data-bs-toggle="tooltip" para el valor estándar de Bootstrap, y no olvides inicializar los tooltips con JavaScript después de cargar la página.
