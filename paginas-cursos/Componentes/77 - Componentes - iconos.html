Iconos en Bootstrap 5: Bootstrap Icons

Bootstrap ofrece su propio set de iconos SVG vectoriales y personalizables a través de la librería Bootstrap Icons. Puedes usarlos en tus botones, menús, formularios, alertas y cualquier elemento HTML de tu sitio.
¿Cómo usar Bootstrap Icons?
  • Solo debes cargar el archivo CSS de Bootstrap Icons (como se muestra en el <head> de esta página).
    <link href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.10.5/font/bootstrap-icons.min.css" rel="stylesheet" />
  • Para mostrar un icono, añade la clase bi y el nombre del icono:
    <i class="bi bi-alarm"></i>
  • Personalización sencilla: tamaño, color, posición, sombra y animaciones con clases de Bootstrap y CSS.

Ejemplo básico: Algunos iconos populares

<i class="bi bi-alarm"></i>
<i class="bi bi-heart-fill text-danger"></i>
<i class="bi bi-github"></i>

Ejemplos útiles: Iconos en botones, inputs y más

Iconos en botones
<button class="btn btn-success"><i class="bi bi-check-circle"></i> Guardar</button>
<button class="btn btn-danger"><i class="bi bi-trash3"></i> Eliminar</button>
Iconos en campos de formulario (inputs)
<div class="input-group">
  <span class="input-group-text"><i class="bi bi-envelope-fill"></i></span>
  <input type="email" class="form-control" ... >
</div>

Personalización de iconos

  • Cambiar tamaño: style="font-size: 2.5em;" o clases utilitarias .fs-1, .fs-2, etc.
  • Cambiar color: Usa clases como text-primary, text-danger, text-success, o estilos CSS.
  • Sombras y fondo: Puedes aplicar .shadow y bg-light para destacar iconos.
  • Animaciones: Algunos iconos pueden animarse con CSS, por ejemplo para indicar carga o alerta.
  • Accesibilidad: Usa aria-label o title para describir el icono.

¿Dónde encontrar más iconos?

Explora la colección completa en icons.getbootstrap.com.
Puedes buscar, copiar el nombre de la clase y pegarlo en tu HTML. Cada icono es un SVG optimizado y puede usarse con cualquier color y tamaño.

Ejemplo avanzado: Iconos interactivos

<button class="btn btn-outline-secondary">
  <i class="bi bi-arrow-clockwise"></i>
</button>
<a href="https://github.com/" class="btn btn-outline-dark">
  <i class="bi bi-github"></i>
</a>
<span style="cursor:pointer;">
  <i class="bi bi-star" id="starIcon"></i>
</span>
TIP: ¡Puedes cambiar el icono dinámicamente con JavaScript!
Ejemplo: Haz clic en la estrella o el corazón para activar/desactivar el favorito o el like.
