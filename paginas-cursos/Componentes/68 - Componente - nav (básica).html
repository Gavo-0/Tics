Curso de Bootstrap - Nav

Menú de navegación (Nav) en Bootstrap 5

Bootstrap 5 proporciona la clase nav para crear menús de navegación horizontales o verticales. Es recomendable usar la etiqueta nav para respetar el concepto semántico de navegación.

  • nav: Clase principal para el menú.
  • nav-link: Clase para cada enlace.
  • disabled: Desactiva el enlace.
  • justify-content-*: Alinea horizontalmente las opciones.
  • flex-column: Muestra las opciones en vertical.

Menú horizontal básico

<nav class="nav">
  <a class="nav-link" href="#">Opción 1</a>
  <a class="nav-link" href="#">Opción 2</a>
  <a class="nav-link" href="#">Opción 3</a>
  <a class="nav-link disabled" href="#">Opción 4</a>
</nav>
Visualización:
Opción 1 Opción 2 Opción 3 Opción 4 Opción 5

Alineación de opciones

Puedes alinear las opciones usando las clases justify-content-end, justify-content-start, justify-content-center, justify-content-between y justify-content-around. 

<nav class="nav justify-content-end">...</nav>
<nav class="nav justify-content-center">...</nav>
<nav class="nav justify-content-between">...</nav>
<nav class="nav justify-content-around">...</nav>

Menú vertical

Usa flex-column para mostrar las opciones una debajo de la otra. 

<nav class="nav flex-column">
  <a class="nav-link" href="#">Opción 1</a>
  <a class="nav-link" href="#">Opción 2</a>
  ...
</nav>

Nota: Puedes combinar estos estilos para crear menús de navegación adaptados a tu sitio web.
