Barras de navegación (Navbar) en Bootstrap 5

Con Bootstrap 5 puedes crear barras de navegación horizontales y responsivas de manera sencilla. Puedes elegir el color de fondo, el color de los enlaces y el comportamiento en dispositivos móviles (colapsar o mostrar vertical).

¿Cómo funciona el responsive?
Al usar navbar-expand-* puedes decidir en qué tamaño de pantalla la barra se colapsa (se muestra vertical).
Por ejemplo, navbar-expand-sm se colapsa en pantallas chicas, navbar-expand-lg en pantallas grandes.
Colores: Cambia el fondo con bg-primary, bg-dark, bg-warning y más. Usa navbar-light para textos oscuros (fondos claros) y navbar-dark para textos claros (fondos oscuros).

Ejemplo para copiar y pegar

<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
  <ul class="navbar-nav">
    <li class="nav-item">
      <a class="nav-link active" href="#">Inicio</a>
    </li>
    <li class="nav-item">
      <a class="nav-link" href="#">Acerca de</a>
    </li>
    <li class="nav-item">
      <a class="nav-link" href="#">Servicios</a>
    </li>
    <li class="nav-item">
      <a class="nav-link" href="#">Contacto</a>
    </li>
  </ul>
</nav>
Consejos rápidos:
  • Cambia bg-primary por cualquier otro color contextual (bg-secondary, bg-info, etc.)
  • Puedes anidar menús, botones, buscadores, íconos y mucho más en la navbar.
  • Combínalo con utilidades de Bootstrap para crear menús atractivos y funcionales.
