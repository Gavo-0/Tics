Tarjetas dimensionadas en Bootstrap 5

Bootstrap 5 nos permite controlar el tamaño y disposición de las tarjetas (.card) de diversas formas, usando tanto el sistema de grillas (.row y .col-*) como clases de anchura o estilos personalizados.

Problema práctico:
  • Mostrar tres tarjetas donde las dos primeras ocupen 5 columnas cada una y la tercera 2 columnas en pantallas extra grandes (col-xxl-*).
  • Si el ancho de la ventana es menor a 1400px, las tarjetas se apilan en una sola columna.

1. Usando grilla Bootstrap (col-xxl-*)

Recomendado para mantener responsividad.
Las tarjetas se dimensionan automáticamente según las columnas asignadas: 
<div class="container">   
  <div class="row">
    <div class="col-xxl-5">
      <div class="card bg-primary text-white mb-3">...</div>
    </div>
    <div class="col-xxl-5">
      <div class="card bg-primary text-white mb-3">...</div>
    </div>
    <div class="col-xxl-2">
      <div class="card bg-primary text-white mb-3">...</div>
    </div>
  </div>
</div>

2. Usando clases de ancho relativo (w-25, w-50, w-75)

Puedes forzar el ancho de una tarjeta con clases utilitarias:
<div class="card w-50">...</div>
Nota: El padre debe tener d-flex para que la anchura se aplique horizontalmente. 
<div class="d-flex gap-3">
  <div class="card w-50">...</div>
  <div class="card w-75">...</div>
  <div class="card w-25">...</div>
</div>

3. Usando una anchura fija por CSS

Puedes establecer un ancho fijo directamente en el estilo:
<div class="card" style="width:20rem">...</div>
<div class="d-flex gap-2">
  <div class="card" style="width:22rem">...</div>
  <div class="card" style="width:18rem">...</div>
  <div class="card" style="width:10rem">...</div>
</div>

4. Igualando la altura de todas las tarjetas

Para que todas las tarjetas tengan la misma altura, añade la clase h-100 a cada tarjeta:
<div class="card h-100">...</div>

Conclusión

  • Usa el sistema de grillas de Bootstrap para un diseño responsivo y fácil de mantener.
  • Para casos puntuales puedes emplear w-25, w-50, w-75 o estilos fijos, pero cuida la responsividad.
  • Para igualar la altura de las tarjetas, usa h-100 en cada una.
