Listas desplegables (Dropdowns) en Bootstrap 5

Las listas desplegables permiten mostrar opciones ocultas que pueden ser seleccionadas por el usuario. Para funcionar correctamente, Bootstrap requiere el archivo bootstrap.bundle.min.js.

  • dropdown: Clase para el contenedor principal del dropdown.
  • dropdown-toggle: Clase para el botón que despliega las opciones.
  • dropdown-menu: Clase para la lista de opciones.
  • dropdown-item: Clase para cada opción individual.

Dropdown básico

<div class="dropdown">
  <button class="btn btn-primary dropdown-toggle" type="button" id="dropdown1" data-bs-toggle="dropdown">
    Buscadores
  </button>
  <div class="dropdown-menu">
    <a class="dropdown-item" href="http://www.google.com">Google</a>
    <a class="dropdown-item" href="http://www.bing.com">Bing</a>
    <a class="dropdown-item" href="http://www.yahoo.com">Yahoo</a>
  </div>
</div>
Visualización:
Google Bing Yahoo

Línea separadora

Para agregar una línea divisoria entre opciones, utiliza dropdown-divider. 

<div class="dropdown-menu">
  <a class="dropdown-item" href="http://www.google.com">Google</a>
  <div class="dropdown-divider"></div>
  <a class="dropdown-item" href="http://www.bing.com">Bing</a>
</div>
Visualización:
Google
Bing Yahoo

Tamaños de dropdown

El tamaño se define por el botón usando btn-sm, btn-lg o el tamaño estándar. 

<button class="btn btn-primary btn-sm dropdown-toggle">...</button>
<button class="btn btn-primary dropdown-toggle">...</button>
<button class="btn btn-primary btn-lg dropdown-toggle">...</button>

Despliegue hacia arriba (dropup)

Usa dropup para desplegar la lista hacia arriba. 

<div class="dropup">
  <button class="btn btn-primary dropdown-toggle">...</button>
  <div class="dropdown-menu">...</div>
</div>
Visualización:
Google
Bing Yahoo

Encabezado informativo

Agrega dropdown-header para mostrar un texto informativo en el menú. 

<div class="dropdown-menu">
  <div class="dropdown-header">Buscadores más populares</div>
  <a class="dropdown-item" href="...">Google</a>
</div>
Visualización:
Buscadores más populares
Google Bing Yahoo

Deshabilitar opciones

Agrega disabled a la opción que no se pueda seleccionar. 

<a class="dropdown-item disabled" href="...">Yahoo</a>
Visualización:
Google Bing Yahoo

Lista desplegable a izquierda o derecha

Usa btn-group dropstart o btn-group dropend para mostrar el menú a la izquierda o derecha. 

<div class="btn-group dropstart">...</div>
<div class="btn-group dropend">...</div>
Izquierda:
Google Bing Yahoo
Derecha:
Google Bing Yahoo

Dropdown con formulario

Puedes poner un formulario dentro del dropdown usando las clases ps-4 pe-4 para el padding horizontal. 

<div class="dropdown">
  <button class="btn btn-primary dropdown-toggle">Login</button>
  <div class="dropdown-menu">
    <form class="ps-4 pe-4">
      <!-- campos del formulario -->
    </form>
  </div>
</div>
Visualización:

Split Button

El split button muestra el menú solo al presionar la flecha a la derecha. 

<div class="btn-group">
  <button type="button" class="btn btn-danger">Buscadores</button>
  <button type="button" class="btn btn-danger dropdown-toggle dropdown-toggle-split" data-bs-toggle="dropdown"></button>
  <div class="dropdown-menu">...</div>
</div>
Visualización:
Google Bing Yahoo

Dropdown oscuro

Usa dropdown-menu-dark para mostrar la lista desplegable en color oscuro. 

<div class="dropdown-menu dropdown-menu-dark">...</div>
Visualización:
Google Bing Yahoo

Nota: Bootstrap 5 te permite personalizar y combinar los estilos de dropdowns para adaptarlos a la experiencia que desees en tu sitio web. Recuerda que, para que los dropdowns funcionen, debes incluir bootstrap.bundle.min.js al final de tu documento.
