Barras de progreso en Bootstrap 5

Las barras de progreso son herramientas visuales para mostrar el avance en una actividad o proceso. Bootstrap 5 las implementa fácilmente utilizando las clases progress y progress-bar.

  • progress: Clase para el contenedor de la barra.
  • progress-bar: Clase para la barra interna que muestra el avance.

Barra de progreso por defecto

<div class="progress">
  <div class="progress-bar" style="width:70%">70%</div>
</div>
Visualización:
70%
La propiedad style="width:70%" indica el porcentaje de avance.

Barra de progreso con colores contextuales

Puedes cambiar el color de la barra usando bg-success, bg-info, bg-warning, bg-danger, etc. 

<div class="progress">
  <div class="progress-bar bg-success" style="width:70%">70%</div>
</div>
Visualización:
70%
80%
90%
95%

Barra de progreso a rayas

Añade progress-bar-striped para mostrar líneas en la barra. 

<div class="progress">
  <div class="progress-bar progress-bar-striped bg-success" style="width:70%">70%</div>
</div>
Visualización:
70%

Barra de progreso animada

Para animar la barra, añade progress-bar-animated junto con progress-bar-striped. 

<div class="progress">
  <div class="progress-bar progress-bar-striped bg-success progress-bar-animated" style="width:70%">70%</div>
</div>
Visualización:
70%

Barra de progreso compuesta

Puedes mostrar varias barras dentro de una sola barra de progreso para indicar distintos estados. 

<div class="progress">
  <div class="progress-bar bg-success" style="width:30%">Correcto</div>
  <div class="progress-bar bg-warning" style="width:30%">Cuidado</div>
  <div class="progress-bar bg-danger" style="width:20%">Peligro</div>
</div>
Visualización:
Correcto
Cuidado
Peligro

Alto de la barra de progreso

Puedes modificar la altura de la barra usando la propiedad height en el div principal. 

<div class="progress" style="height:2rem">
  <div class="progress-bar bg-info" style="width:65%">65%</div>
</div>
Visualización:
65%

Ejemplo avanzado: Combinando estilos

<div class="progress mb-3" style="height:2rem">
  <div class="progress-bar progress-bar-striped bg-success progress-bar-animated" style="width:70%">70%</div>
</div>
Visualización:
70%

Nota: Puedes combinar los diferentes estilos para mostrar el avance de procesos con la apariencia más adecuada a tu aplicación.
