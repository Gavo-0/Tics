Nav-tabs y Nav-pills en Bootstrap 5

Puedes transformar tu menú de navegación en pestañas (nav-tabs) o en pastillas (nav-pills) de forma sencilla. Estas variantes mejoran la presentación y experiencia de navegación en tu sitio web.

  • nav-tabs: Muestra las opciones como pestañas.
  • nav-pills: Muestra las opciones como pastillas/redondeadas.
  • active: Marca la opción activa.
  • disabled: Desactiva una opción.
  • nav-justified: Las opciones ocupan todo el ancho del contenedor.
  • nav-item: Usar en cada opción junto con nav-justified para igualar el tamaño.
  • mt-2: Agrega margen superior a la barra de navegación.

Menú con pestañas (nav-tabs)

<nav class="nav nav-tabs mt-2">
  <a class="nav-link" href="#">Opción 1</a>
  <a class="nav-link active" href="#">Opción 2</a>
  <a class="nav-link" href="#">Opción 3</a>
  <a class="nav-link" href="#">Opción 4</a>
  <a class="nav-link disabled" href="#">Opción 5</a>
</nav>
Visualización:
Opción 1 Opción 2 Opción 3 Opción 4 Opción 5
Las pestañas permiten al usuario identificar qué opción está activa mediante la clase active.

Menú con pastillas (nav-pills)

<nav class="nav nav-pills mt-2">
  <a class="nav-link" href="#">Opción 1</a>
  <a class="nav-link active" href="#">Opción 2</a>
  <a class="nav-link" href="#">Opción 3</a>
  <a class="nav-link" href="#">Opción 4</a>
  <a class="nav-link disabled" href="#">Opción 5</a>
</nav>
Visualización:
Opción 1 Opción 2 Opción 3 Opción 4 Opción 5
Las opciones aparecen como botones redondeados (pastillas), útiles para menús de secciones o filtros.

Opciones ocupando todo el ancho (nav-justified)

Para que las opciones ocupen todo el ancho del contenedor, usa nav-justified en el nav y nav-item en cada enlace. 

<nav class="nav nav-pills nav-justified mt-2">
  <a class="nav-item nav-link" href="#">Opción 1</a>
  <a class="nav-item nav-link active" href="#">Opción 2</a>
  <a class="nav-item nav-link" href="#">Opción 3</a>
  <a class="nav-item nav-link" href="#">Opción 4</a>
  <a class="nav-item nav-link disabled" href="#">Opción 5</a>
</nav>
Visualización:
Opción 1 Opción 2 Opción 3 Opción 4 Opción 5
Cada opción se ajusta automáticamente para ocupar el mismo espacio, útil en menús principales.

Colores personalizados en nav-pills

Puedes aplicar clases de color (bg-primary, bg-success, etc.) para destacar la opción activa, usando text-white si es necesario.
