Navbar responsiva y botón hamburguesa en Bootstrap 5

Bootstrap 5 facilita la creación de barras de navegación que se adaptan a cualquier tamaño de pantalla.
  • En pantallas grandes, la barra muestra todas las opciones horizontalmente.
  • En dispositivos angostos, muestra un botón hamburguesa que despliega el menú al hacer clic.

¿Cómo funciona?
  • La clase navbar-expand-sm indica que será horizontal desde pantallas pequeñas hacia arriba. En pantallas menores, colapsa y muestra el botón hamburguesa.
  • El navbar-toggler es el botón que despliega el menú colapsado.
  • El menú colapsable debe tener un id único (ejemplo: opciones) y el botón debe tener data-bs-target="#opciones".
  • Puedes incluir un logo o nombre de marca con navbar-brand.

Ejemplo para copiar y pegar

<nav class="navbar navbar-expand-md navbar-dark bg-primary">
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#menuNav">
    <span class="navbar-toggler-icon"></span>
  </button>
  <a class="navbar-brand ms-2" href="#">
    <img src="https://www.tutorialesprogramacionya.com/imagenes/foto1.jpg" width="30" height="30" alt="Logo">
  </a>
  <div class="collapse navbar-collapse" id="menuNav">
    <ul class="navbar-nav">
      <li class="nav-item">
        <a class="nav-link active" href="#">Inicio</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Servicios</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Contacto</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Ayuda</a>
      </li>
    </ul>
  </div>
</nav>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0/dist/js/bootstrap.bundle.min.js"></script>
Tips rápidos:
  • Puedes cambiar navbar-expand-md por sm, lg, xl según cuándo quieres que colapse.
  • Personaliza colores usando bg-dark, bg-success, etc., y combina con navbar-dark o navbar-light.
  • Puedes poner el botón hamburguesa a la derecha agregando ms-auto a la marca o menú.
