Badges (Distintivos) en Bootstrap 5

Un badge es una pequeña etiqueta que se usa para mostrar mensajes cortos, contadores, o destacar información en otros componentes como botones, encabezados o menús.

  • badge: Clase principal del distintivo.
  • bg-*: Color contextual del badge (ej: bg-warning, bg-danger).
  • ms-1: Margen izquierdo para separar el badge del texto contiguo.
  • rounded-pill: Vértices redondeados, forma de píldora.

Badge en un título

<h1>Tutorial de Bootstrap 5 <span class="badge bg-warning">Nuevo</span></h1>

Badge en un botón

<button type="button" class="btn btn-primary">
  Mensajes<span class="badge bg-danger ms-1">4</span>
</button>
Visualización:
Usa ms-1 para separar el badge del texto del botón.

Badges en forma de píldora

Agrega la clase rounded-pill para que el badge tenga vértices redondeados. 

<span class="badge rounded-pill bg-warning">Nuevo</span>

Badge en un accordion

<button class="accordion-button">
  Mensaje <span class="badge bg-success ms-2">¡Nuevo!</span>
</button>

Colores contextuales para badges

<span class="badge bg-primary">Primary</span>
<span class="badge bg-secondary">Secondary</span>
<span class="badge bg-success">Success</span>
<span class="badge bg-danger">Danger</span>
<span class="badge bg-warning text-dark">Warning</span>
<span class="badge bg-info text-dark">Info</span>
<span class="badge bg-dark">Dark</span>
<span class="badge bg-light text-dark">Light</span>
Visualización: Primary Secondary Success Danger Warning Info Dark Light

Nota: Los badges permiten destacar información importante y pueden combinarse con otros componentes de Bootstrap. Usa rounded-pill para estilo de píldora y bg-* para colores contextuales.
