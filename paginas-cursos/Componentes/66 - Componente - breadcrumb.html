Curso de Bootstrap - Breadcrumb

Breadcrumb (Migas de pan) en Bootstrap 5

Un breadcrumb muestra una lista de enlaces jerárquicos que indican el camino desde la página principal hasta la página actual. Es útil para mejorar la navegación y la experiencia del usuario.

  • breadcrumb: Clase para la lista ordenada (ol).
  • breadcrumb-item: Clase para cada elemento (li).
  • active: Clase para el elemento actual (sin enlace).

Ejemplo básico de breadcrumb

<ol class="breadcrumb">
  <li class="breadcrumb-item"><a href="#">Diseño web</a></li>
  <li class="breadcrumb-item"><a href="#">CSS</a></li>
  <li class="breadcrumb-item active">Bootstrap 5</li>
</ol>
Visualización:
  1. Diseño web
  2. CSS
  3. Bootstrap 5
El último elemento es la sección actual y no tiene enlace. Los anteriores sí permiten navegar.

Personalizar el separador

Puedes cambiar el separador usando la variable --bs-breadcrumb-divider en el nav que contiene el breadcrumb. 

<nav style="--bs-breadcrumb-divider: '>'">
  <ol class="breadcrumb">...</ol>
</nav>
Visualización:
  1. Diseño web
  2. CSS
  3. Bootstrap 5
El separador será el caracter > en lugar del predeterminado.

Ejemplo avanzado: Breadcrumb con más niveles

<nav style="--bs-breadcrumb-divider: '|'">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Inicio</a></li>
    <li class="breadcrumb-item"><a href="#">Lenguajes</a></li>
    <li class="breadcrumb-item"><a href="#">Frontend</a></li>
    <li class="breadcrumb-item active">Bootstrap 5</li>
  </ol>
</nav>
Visualización:
  1. Inicio
  2. Lenguajes
  3. Frontend
  4. Bootstrap 5

Nota: Utiliza breadcrumbs para ayudar al usuario a entender la jerarquía y facilitar la navegación en tu sitio web.
