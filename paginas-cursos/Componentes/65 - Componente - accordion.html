Curso de Bootstrap - Accordion

Accordion en Bootstrap 5

El componente accordion permite mostrar varias áreas de contenido oculto que se despliegan al presionar sus títulos. Es ideal para preguntas frecuentes, políticas, características de productos, etc.

  • accordion-item: Define cada sección del acordeón.
  • accordion-header: Cabecera con el botón que despliega/cierra el contenido.
  • accordion-button: Botón para mostrar/ocultar el contenido.
  • accordion-collapse collapse: Área de contenido oculto.
  • show: Hace que el contenido esté visible al cargar.
  • data-bs-parent: Permite que sólo una sección esté abierta a la vez.

Ejemplo básico de accordion

<div id="accordion1">
  <div class="accordion-item">
    <div class="accordion-header" id="cabecera1">
      <button class="accordion-button" data-bs-toggle="collapse" data-bs-target="#respuesta1"
        aria-expanded="true" aria-controls="respuesta1">
        Pregunta frecuente 1
      </button>
    </div>
    <div id="respuesta1" class="accordion-collapse collapse show"
      aria-labelledby="cabecera1" data-bs-parent="#accordion1">
      <div class="accordion-body">...</div>
    </div>
  </div>
  ... 
</div>

El atributo data-bs-parent="#accordion1" asegura que solo una sección esté abierta a la vez.

Accordion con contenido abierto sin cerrar los demás

Si omites data-bs-parent en cada accordion-collapse, puedes abrir varias áreas a la vez. 

<div id="respuesta1" class="accordion-collapse collapse show"
  aria-labelledby="cabecera1">
  ...
</div>
Visualización:
Este contenido está abierto al cargar y puedes abrir los demás sin cerrarlo.
Puedes abrir este contenido sin cerrar el anterior.
Omite data-bs-parent para permitir múltiples áreas abiertas simultáneamente.

Nota: Puedes personalizar los títulos y el contenido de cada área según tus necesidades. Recuerda incluir bootstrap.bundle.min.js para que la funcionalidad del accordion funcione correctamente.
