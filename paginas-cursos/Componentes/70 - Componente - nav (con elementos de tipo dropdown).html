Menú horizontal con opción Dropdown en Bootstrap 5

Puedes agregar un menú desplegable (dropdown) dentro de un menú horizontal creado con la clase nav. Es útil para mostrar subopciones dentro de tu barra de navegación.

  • ul.nav: Estructura principal del menú.
  • li.nav-item: Cada opción del menú.
  • a.nav-link: Hipervínculo de cada opción.
  • li.nav-item.dropdown: Opción que contiene el dropdown.
  • a.nav-link.dropdown-toggle: Enlace que despliega el dropdown.
  • div.dropdown-menu: Contenedor de las opciones desplegables.
  • a.dropdown-item: Cada opción dentro del dropdown.

Ejemplo de menú horizontal con dropdown

<ul class="nav mt-2">
  <li class="nav-item">
    <a class="nav-link" href="#">Opción 1</a>
  </li>
  <li class="nav-item">
    <a class="nav-link active" href="#">Opción 2</a>
  </li>
  <li class="nav-item dropdown">
    <a class="nav-link dropdown-toggle" data-bs-toggle="dropdown" href="#">Buscadores</a>
    <div class="dropdown-menu">
      <a class="dropdown-item" href="http://www.google.com">Google</a>
      <a class="dropdown-item" href="http://www.bing.com">Bing</a>
      <a class="dropdown-item" href="http://www.yahoo.com">Yahoo</a>
    </div>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Opción 4</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#">Opción 5</a>
  </li>
</ul>
Visualización:
  • Opción 1
  • Opción 2
  • Buscadores
    Google Bing Yahoo
  • Opción 4
  • Opción 5

Nota: Puedes agregar varias opciones de tipo dropdown en tu menú horizontal. Recuerda incluir bootstrap.bundle.min.js para que los dropdowns funcionen correctamente.
